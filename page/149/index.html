Gem安装rubyrails的方法

2020年9月2日
使用gem install rails命令并不能顺利的安装，连接失败！ 下面有两个解决方法： 1.离线安装 在网上找了很多帖子都没有找到直接可用的方法，因此直接用gem –help 查看了安装方法，可以使用 –local 参数，进行离线安装。首先去官网下载原安装包，然后进行离线安装。 step1:先下载依赖安装包文件 http://rubygems.org/downloads/rails-4.2.0.gem http://rubygems.org/downloads/activesupport-4.2.0.gem http://rubygems.org/downloads/activerecord-4.2.0.gem http://rubygems.org/downloads/actionpack-4.2.0.gem http://rubygems.org/downloads/actionmailer-4.2.0.gem step2:本地离线安装 gem install –local rails-4.2.0.gem 2.替换gem源为淘宝源，在线安装 gem 在线安装失败的原因，gem源本身不稳定，可以替换为其它源比如taobao.org就可以。 具体方法如下： 删除默认的官方源 gem sources -r https://rubygems.org/ 添加淘宝源 gem sources -a https://ruby.taobao.org/ 查看当前源 gem sources -l 返回结果 https://ruby.taobao.org 这样替换源就换号了，再次使用gem install rails 就可以正确安装rails啦。……

利用shell脚本一键安装mysqlrpm包

2020年9月2日
mysql的安装包一般我们用两种 第一种就是rpm包的安装，这个安装比较简单，不需要编译， 第二种是源码二进制glibc的安装，相对第一种来说比较繁琐，更容易出差错 接下来，我用shell脚本来实现mysql的这两种安装 首先实现第一种 我安装的是5.7.14版本的rpm包 先写shell脚本 在/opt/scripts目录下编写脚本rpm_install.sh (目录任意，脚本名任意) #!/bin/bash #安装mysql的rpm包 #编写日期：2019-04-01 #需要提前配置好yum源并将rpm包放到/root目录下 --------- 一定要做的操作 #在/root目录下创建一个文件夹用来存放rpm的tar包解压后的rpm包 cd /root mkdir mysql #在/root目录下将rpm包解压到mysql目录下 tar -xvf mysql-5.7.14-1.el6.x86_64.rpm-bundle.tar -C mysql #解压比较慢 #在mysql目录下，安装所有rpm包 cd mysql yum localinstall ./mysql-community-*.rpm -y #启动mysql服务 service mysqld start #------------------------------------------------------ #以下内容手工执行 #查找临时密码 #cat /var/log/mysqld.log | grep "temporary password" #用临时密码进行登录mysql数据库（注意：临时密码一定要放到双引号里面括起来，也可以用单引号，防止登录不成功） （双引号登录不成功用单引号登录） #mysql -uroot -p"临时密码" # 修改密码 #set password=password('新密码'); （注意：新密码要复杂一点，要不然修改不成功） #退出mysql数据库并重新登录 #quit #mysql -uroot -p新密码 然后在/opt/scripts目录下执行脚本即可 bash rpm_install.sh 此时mysql服务已经启动，接下来，手工执行脚本里面用#号注释掉的命令即可。……

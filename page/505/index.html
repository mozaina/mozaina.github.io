mac安装homebrew配置rubyecurlinsecurefsSLhttpsrawgithubusercontentcomHomebrewinstal

2020年9月2日
1. 打开 Mac 中的命令行工具（终端） 2. 在打开的命令行工具中输入如下语句： ruby -e "$(curl --insecure -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" //注意 ruby -e 之间的空格 3. 执行上面命令后会提示输入系统密码 4. 输入密码后会开始下载homebrew,如果此时网速不稳定可能会出现错误提示：curl: (35) Server aborted the SSL handshake。 此时需要执行卸载操作，输入如下命令进行卸载： ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/uninstall)" 卸载完成后再次尝试安装。 5. 安装完成后会有提示，有两种提示： 一种是 警告 提示，如下： 出现上述 红色提醒，解决方法，只需打开 PATH 配置文件，加入以下路径命令，操作具体如下： 1 打开 PATH 配置文件，使用快键键或直接前往文件夹 /etc/profile，打开 profile 文件 2 添加路径语句：export PATH=/usr/local/bin:$PATH 点击保存即可 已添加上述路径，则会出现安装成功的提示，如下： 6. 此时可以在命令行中输入命令 brew 进行测试，可以看到如下结果,说明安装成功，也可使用 brew –version 查看安装的版本信息等。……

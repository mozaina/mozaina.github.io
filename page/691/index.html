Ruby命令之gem操作

2020年9月2日
1. 查看已安装的gem： gem list 2. 在服务器上查询gem： gem query -remote 简写为：gem q -R 3. 在线安装gem： gem install gem_name 若要指定版本： gem install gem_name –version=1.2.3 4. 离线安装gem： gem install ./gem_name –local 运行命令时要在gem所在的目录，gem下载地址：http://rubygems.org/ 或者：gem install -l gem_name.gem 5. 卸载gem： gem uninstall gem_name gem uninstall gem_name –version=1.2.3 6. 更新所有gem： gem update 更新指定的gem： gem update gem_name……

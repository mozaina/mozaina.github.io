利用shell脚本按条件过滤文档判断是否参数是整数

2020年9月2日
需求：过滤第一行参数超过7000的行数 文档内容如下： #cat ./result.txt sshuser=5.5.5.5 8000 1.1.1.1 7000 2.2.2.2 脚本关键点有两个，首先进行判断参数是否是整数，其次是输出文件中 #cat edit-result.sh #!/bin/bash cat ./result.txt|while read readingline do countnum=echo $readingline|awk '{print $1}' expr $countnum + 0 &>/dev/null if [ $? -ne 0 ];then echo $readingline>>result2.txt fi if [ $? -eq 0 ] && [ $countnum -gt 7100 ];then echo $readingline>>result2.txt fi done 这样就可以完成预想功能了……

shell中后台运行函数

2020年9月2日
在shell中定义好的函数,运行的时候可以直接在脚本里面通过func &方式后台运行。 一开始我很好奇为什么shell脚本里面定义好的函数可以这么用，通常我们在脚本中都是以后台方式运行其它的脚本或者命令。 于是做了如下一个实验： new() { echo "func bkground pid by \$\$ is $$" while [ 1 == 1 ] do sleep 5 done } echo "current script pid is $$" new & while [ 1 == 1 ] do sleep 5 done 运行如上脚本后，通过ps查看会发现当前运行了两个shell进程，输出如下： [root@localhost ~]# ps -ef|grep test root 26960 30678 0 12:05 pts/0 00:00:00 sh test.sh root 26961 26960 0 12:05 pts/0 00:00:00 sh test.sh 所以可以确认，shell里面是可以直接以后台方式运行函数的，后台运行的函数以一个新的进程运行。……

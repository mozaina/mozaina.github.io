1,什么是shell脚本？ shell脚本是在linux下运行的文件。文件内容是一系列可执行命令。 2，怎么开始写shell脚本？ 1，安装ubuntu系统或者有可供连接的远程linux系统 2，编写脚本 3，给脚本赋予可执行权限 脚本文件格式 为了保持编程传统，我们将创建一个“hello world”程序来说明一个极端简单的脚本。脚本内容： #! /bin/bash # this is hello shell comment echo 'hello world' 第一行不是注释。“#！”是操作系统将执行次脚本所用的解释器的名字。每个shell脚本都应该把这一文本行作为它的第一行。 第二行是注释。 第三行是在屏幕上打印 hello world 编写完以上内容后保存，立即执行会命令会报命令找不到错误。需要给文件赋予可执行权限，执行命令如下： chmod 777 shell_hello.sh 或 chmod +x shell_hello.sh 后缀名sh可有可无的，linux中后缀名只是一种符号，但是在windows下用习惯了，不加后缀名怪怪的，所以就在创建的时候以.sh结尾。 执行的时候，直接执行 shell_hello.sh可能会报错，因为linux会默认查找path目录下，如果shell_hello.sh没在path目录就会报错。如果是当前目录执行，可以直接用./shell_hello.sh执行。 整个过程命令和输出截图： 查看path目录可以直接在命令行输入$PATH,如图：……

hbase启动成功却不能web端口访问也无法使用hbaseshell

2020年9月2日
HTTP ERROR 503 求救 访问HBase的WebUI(60010端口)时，一致出现 HTTP ERROR 503 Problem accessing /master-status.Reason: master not ready 原因1：如果hdfs是HA模式且namenode节点都是standby的话，就算启动Hmaster也无法从web端查看监控情况，因为hdfs在standby状态下是无法提供服务的，因为hbase也是基于hdfs服务的。需要启动zkfc即可。 原因2：hdfs是安全模式 使用 $bin/hadoop dfsadmin -safemode leave 也就是关闭Hadoop的安全模式,然后就可以访问了……

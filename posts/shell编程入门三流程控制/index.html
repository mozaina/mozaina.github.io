Shell编程入门三流程控制

2020年9月2日
| 阅读

和Java、PHP等语言不一样，sh的流程控制不可为空，如：

public class Test{
    public static void main(String[] args){
        int a = 10;
        if(a > 5){
            System.out.println("a 大于 5");
        }
        else{
            //不做任何事
        }
    }
}

在bash里不能这么写，如果 else 分支没有语句执行，就不要写这个 else 。

if

语句语法格式如下：

a=10
b=20
if [ $a == $b ]
then
    echo "a is same to b."
fi

if else

语句语法格式如下：

a=10
b=20
if [ $a == $b ]
then
    echo "a is same to b."
else
    echo "a is not same to b."
fi

if else-if else

语句语法格式如下：

a=10
b=20
if [ $a == $b ]
then
   echo "a 等于 b"
elif [ $a -gt $b ]
then
   echo "a 大于 b"
elif [ $a -lt $b ]
then
   echo "a 小于 b"
else
   echo "没有符合的条件"
fi

输出结果为：a 小于 b。

if else语句经常与test命令结合使用，如下所示：

num1=$[2*3]
num2=$[1+5]
if test $[num1] -eq $[num2]
then
    echo '两个数字相等!'
else
    echo '两个数字不相等!'
fi

输出结果为：两个数字不相等。

for 循环

顺序输出当前列表中的数字：

for loop in 1 2 3 4 5
do
    echo "The value is : $loop"
done

while 语句

num=1
while(( $num<=5 ))
do
    echo $num
    let "num++"
done

在bash中使用了 let 命令，它用于执行一个或多个表达式。变量计算中不需要加上 $ 变量。

while 循环可用于读取键盘信息。以下例子，输入信息被设置为变量 input ，按 Ctrl + D 结束循环。

echo "按下 ctrl + D 退出"
echo "请输入信息："
while read input
do
    echo "your input is : $input"
done

until 循环

until 循环执行一系列命令直至条件为真时停止。
until 循环与 while 循环在处理方式上刚好相反。

num=5
until(( $num>10 ))
do
    echo $num
    let "num++"
done

结果将依次输出：5 6 7 8 9 10

case

可以用 case 语句匹配一个值与一个模式，如果匹配成功，执行相匹配的命令。

echo '请输入信息'
read aNum
case $aNum in
    1)  echo '你选择了 1'
    ;;
    2)  echo '你选择了 2'
    ;;
    3)  echo '你选择了 3'
    ;;
    "wsz")  echo '你选择了 wsz'
    ;;
    *)  echo '你没有输入 1 到 4 之间的数字'
    ;;
esac
  • case 工作方式如上所示。取值后面必须为单词 in ，每一模式必须以右括号结束。取值可以为变量或常数。匹配发现取值符合某一模式后，其间所有命令开始执行直至 ;; 表示 break。
  • 取值将检测匹配的每一个模式。一旦模式匹配，则执行完匹配模式相应命令后不再继续其他模式。如果无一匹配模式，使用星号 * 捕获该值，再执行后面的命令。
  • esac 语法和C family语言差别很大，它需要一个 esac（就是case反过来）作为结束标记，每个 case 分支用) ，用 ;; 表示 break。

跳出循环

break

下面的例子中，脚本进入死循环直至用户输入数字不是 1 2 3 4 5 中的一个。要跳出这个循环，需要使用break命令。

while :
do
    read aNum
    case $aNum in
    1|2|3|4|5) echo "your input is $aNum"
    ;;
    *) echo "game over"
        break
    ;;
    esac
done

continue

continue 命令与 break 命令类似，只有一点差别，它不会跳出所有循环，仅仅跳出当前循环。

for (( i=0 ; i<10 ; i++ ))
do
    if [ $i == 3 ]
    then
        continue
    else
        echo $i
    fi
done

上述结果将输出：0 1 2 4 5 6 7 8 9，不会输出 3。

shell中算术运算符

2020年9月2日
| 阅读

1.shell中的算术运算符和c语言中差不多，优先级和顺序也相同；如下表（其中优先级由高至低）

运算符 意义 顺序
++ - - 增加及减少，可前置/可结尾 由左至右
+ 一元正号 由右至左
- 一元负号 由右手至左
！ 逻辑与 由右手至左
~ 位的取反 由右手至左
* / % 乘法、除法、取余数 由左至右
+ 加法 由左至右
- 减法 由左至右
<< >> 向左位移、向右位移 由左至右
< <= > >= 比较运算符 由左至右
== != 相等 不相等 由左至右
& 位的AND 由左至右
^ 位的Exclusive OR 由左至右
位的OR 由左至右
&& 逻辑AND 由左至右
逻辑OR
?: 条件表达式 由右至左
= += -= *= /= %= &= ^= 赋值运算符 由右至左

备注：可以用圆括号来将子表达式语句块括起来。如c一样，关系运算符（、>=、==、！=）产生的数字结果中，1表示为真，0表示为假。

例1

#########################################################################
# File Name: algorithm.sh
# Author: lixiaogang
# mail: 2412799512@qq.com
# Created Time: 2018年03月13日 星期二 01时55分18秒
#########################################################################
#!/bin/bash
a=3
b=4
if [ $a > $b ]
then
    echo "a > b"
else
    echo "a <= b"
fi

结果：
a > b

例2

#########################################################################
# File Name: unary.sh
# Author: lixiaogang
# mail: 2412799512@qq.com
# Created Time: 2018年03月13日 星期二 02时03分15秒
#########################################################################
#!/bin/bash
var=5
echo $((var++)) $var            # 5 6
echo $((++var)) $var            # 7 7

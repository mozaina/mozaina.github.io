查看文件树形结构

yum install tree -y #全自动选y
tree #树形结构去查看文件

ifconfig -a #查询服务器的IP

Linux 只有目录，没有文件夹

连接服务器的工具

1.putty（免费）推荐官网下载

输入IP 端口号即可

输入用户名密码进行登录

2.Xshell(收费) 推荐使用个人版

新建会话

拥有Xftp用于文件传输

其他文件传输工具 lrzsz

rz #上传

sz 文件名  #下载

[用户名@主机名 当前目录]$/# $普通用户 # 为ROOT用户

useradd zpcs01 #增加用户zpcs01

passwd zpcs01 #设置zpcs01密码

su zpcs01 #切换zpcs01用户

history #查看历史命令

history -c #清楚历史命令

clear #清屏

uname -a #查看服务器型号

pwd #显示当前绝对路径

reboot #重启

halt #关机

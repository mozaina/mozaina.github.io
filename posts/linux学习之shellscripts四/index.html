一、不定循环：while do done,until do done

while [condition] 
do
    程序段落
done


-------------------------------------------------------------------------------------

until [condition]
do
    程序段落
done

eg1: 1+2+3+4+5+…….+100

[dalianmao@localhost ~]$ sh caculate.sh 
1+2+3+4+5+...+100= 5050
[dalianmao@localhost ~]$ nl caculate.sh 
     1  #!/bin/bash

     2  #Program:
     3  #      This program caculate 1+2+3+4+....+100


     4  i=1
     5  s=0

     6  while [ $i -le 100 ]

     7  do

     8      s=$(($s+$i))
     9      i=$(($i+1))

    10  done

    11  echo "1+2+3+4+5+...+100=" $s

eg2:1+2+3+4+5+…..+100

[dalianmao@localhost ~]$ sh caculate_01.sh 
1+2+3+4+5+...+100= 5050
[dalianmao@localhost ~]$ nl caculate_01.sh
     1  #!/bin/bash

     2  #Program:
     3  #      This program caculate 1+2+3+4+....+100


     4  i=1
     5  s=0

     6  until  [ $i -gt 100 ]

     7  do

     8      s=$(($s+$i))
     9      i=$(($i+1))

    10  done

    11  echo "1+2+3+4+5+...+100=" $s

二、固定循环：for ….do….done

（1）相对while，until的循环方式是必须符合某个条件的状态，用for时一般已经知道要进行几次循环。

for var in con1 con2 con3 con4 con5.....
do
    程序段
done

eg1.循环输出账号信息

[root@localhost dalianmao]# nl check_account.sh 
     1  #!/bin/bash

     2  #Program:
     3  #       This program check the information of account

     4  username=$(cut -d':' -f1 /etc/passwd)

     5  for var in $username
     6  do
     7      id $var
     8      finger $var
     9      echo "==============================================================="
    10  done
    11  echo ">>>>>>>>>>>>>>>>>>>>>>end<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<>>>>>>>>>>>>>>>>>>>>>end<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<

（2）for …do……done :数值处理

for ((初始值;限制值;执行步阶))
do
    程序段
done


初始值：某个变量在循环当中的起始值，直接以类似i=1设定好；
限制值：当变量的值在这个限制值得范围内，就继续进行循环。例如i<100;
执行步阶：每做一次循环时，变量的变化量，例如i=i+1

eg:1+2+3+4+5+…

[root@localhost dalianmao]# sh caculate_02.sh 
please input number....100
1+2+3+...+100= 5050
[root@localhost dalianmao]# nl caculate_02.sh 
     1  #!/bin/bash

     2  #Program:

     3  #     This program will caculate 1+2+3+4+....$num



     4  read -p "please input number...." num

     5  for ((i=1;i<= $num;i=i+1))

     6  do

     7      s=$(($s+i))

     8  done

     9  echo "1+2+3+...+$num=" $s

function fname{

eg:两数相加

input second number:12

0

     2  #Program:
     3  #       This program test function



     4  function  number_plus(){
     5      result=$(($1+$2))
     6      echo $result
     7      return 0


     8  }



     9  read -p "input first number:" num1

    10  read -p "input second number:" num2

    11  number_plus $num1 $num2
    12  echo $?

注： shell函数中，return值只能是整数常量，如果要’返回’,可以把函数内的变量设为全局变量

圣诞树闪、颜色:

\033[代表格式,后面的数字是属性,数字之间用分号隔开,m是结束的标志

\033[格式的属性介绍如下：

5闪烁,0关闭所有属性,1高亮

前景色：黑30，红31，绿32，黄33，蓝34，洋红35，青36，白37

背景色：黑40，红41，绿42，黄43，蓝44，洋红45，青46，白47

echo -e”\033[5m*******************************\033[0m”

echo -e”\033[5;33;1m*******************************\033[0m”

echo “libanxian”

注释的小技巧：按ctrl+v进入可视化块,选中第一列，按大写I，输入#，按两下ESC可以添加注释，去掉注释的话：ctrl+v进入可视化块，选中第一列，按x删除，或者进入命令行模式输入:3,8 s/^/#/g

和3,8 s/^#//g

挂起虚拟机之后再睡眠电脑，要不然会出现ll无效、中文变成问号、执行文件没有绿色的问题

ll无效解决方案、执行文件没有绿色：设置别名：alias ll=“ls-l –color=auto”

alias ls=“ls –color=auto”

中文变成问号解决方案：(1) 执行locale命令查看系统语言(2)设置系统环境变量LANG为en_US.UTF-8： exportLANG=en_US.UTF-8，要想永久生效，需要vim /etc/profile回车，export LANG=en_US.UTF-8保存退出，然后执行source/etc/profile即可，zh_CN.UTF-8才是中文，en_US的是英中文

2.输入j打印1～100之间的奇数相加之和，输入o打印1～100之间的偶数之和。

解：

read -p “输入j求1~100奇数之和，输入o求1~100偶数之和:” input

sum=0

if [ $input == “j” ];then

for((i=1;i<=100;i++));do

if [ $[${i}%2] == 1 ];then

jishu=$i

sum=$[${jishu}+${sum}]

fi

done

echo $sum

elif [ $input == “o” ];then

for((i=1;i<=100;i++));do

if [ $[${i}%2] == 0 ];then

oushu=$i

sum=$[${oushu}+${sum}]

fi

done

echo $sum

fi

b=0

#清空up和down里面的内容

ls /tmp/up.txt &>/dev/null && > /tmp/up.txt

ls /tmp/down.txt &>/dev/null && > /tmp/down.txt

for((i=1;i<=255;i++));do

b=${a}.${i}

if [ $(ping -w 2 $b | grep “loss”| cut -d” “ -f6) == “0%” ];then

else

fi

done

done

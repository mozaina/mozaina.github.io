2020年9月2日
| 阅读

1.输入

read是一个内置的命令，程序执行过程中，遇到read命令就会暂停执行，等待用户键盘的输入，当用户输入数据并按回车键后，将完成赋值操作，程序继续运行。

下面看一个具体的例子，等待用户输入用户名，并在用户名前添加hello输出：

脚本：
#!/bin/bash

echo "请输入你的名字"
read NAME
echo "hello $NAME "


执行：
bash hello_user.sh

输出：
请输入你的名字
jack
hello jack

read 命令还可以一次读取多个值，并赋值给相应变量

  • 多个数据和变量之间用空格分隔
  • 若变量个数与数据个数相等，则对应赋值
  • 变量个数大于输入数据个数，多余的变量为空值
  • 变量个数少于数据个数，多余的数据赋值给最后一个变量

具体的例子：

脚本：
#!/bin/bash

echo "请输入多个用户名"
read USER1 USER2
echo "hello $USER1"
echo "hello $USER2"

执行：
bash hello_users.sh

输出：
请输入多个用户名
JACK ROBINSON ALICE 
hello JACK
hello ROBINSON ALICE

#

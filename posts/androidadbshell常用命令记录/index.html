androidadbshell常用命令记录

2020年9月2日
| 阅读

apk包安装步骤：
1，首先将../android-sdk/platform/tools添加都path路径下
2，在控制台窗口中进入到你apk包所在的目录中
3，输入：abd unremount,第一次使用该命令的时候需要获取该命令的操作权限
4，打开android的模拟器
5，在控制台中执行：adb install 包名.apk
6，如果显示success，那么就安装apk包成功，显示Failure，则安装失败。
7，检查错误，排错，再执行如下命令：abd install -r 包名.apk
8，如果安装成功后，就可以在模拟器中执行你的应用程序

apk安装位置：

1、系统自带的应用apk存储在(安装在)system/app目录下

2、用户安装的应用apk存储在(安装在)data/app目录下 命名规则：APK包名-1。比如：com.simon.view-1.apk

adb常用命令：
1、adb uninstall 包名 —- 卸载一个应用
2、adb install apk文件名称.apk —- 安装一个应用
3、adb install -s apk文件名称.apk —- 安装一个应用到sd卡中
4、adb install -r apk文件名称.apk —- 重新安装该应用

5、adb shell —- 进入手机的超级终端

6、adb reboot —- 重启手机

7、adb logcat -s tag —- 查看指定tag的日志，此处的tag就类似于Eclipse中LogCag的tag

8、adb logcat -s tag -> d:/a.txt —- 将查看指定tag的日志保存到文件中

9、cls —- 清空CMD命令

10、adb push 001.jpg /sdcard —- 复制001.jpg文件到sd卡目录下(注意：原文件的路径根据自己情况指定)
11、adb kill-service \ adb start-service —- 停止和启动abd服务
12、rm xxxx.apk —- 在adb shell 终端下，进入system/app或者data/app目录下删除系统或者自己安装的应用也可以

13、adb logcat -s tag —- adb 查看日志，此处的tag就类似于Eclipse中LogCat的tag

14、adb logcat -s tag ->d:a.txt —- adb 将查看的日志保存到txt

15、adb shell monkey -p 包名 -v 50000 —- adb 压力测试(直接显示在cmd输出里面)

16、adb shell monkey -p 包名 -v 50000 -t >D:monkey.txt —- adb 压力测试(将日志输出到文件里面)

例如：adb shell monkey -p com.huawei.android.mediawork -v 5000 -t >D:monkey.txt

17、adb shell input text http://…… —- 通过命令输入内容

18、adb shell am start -n com.huawei.mediawork.ott/com.huawei.mediawork.activity.SettingActivity —– 启动指定activity

19、adb shell tcpdump -i wlan0(any) -s 0 -X -vv -n -w /data/sock.cap —- 抓网络数据包

20、adb shell am start -a android.intent.action.VIEW -d “http://www.baidu.com" —- 调用系统浏览器打开指定网页
21、adb shell am start -a android.intent.action.VIEW -t video/mp4 -d xxx.ts —- 调用系统播放器播放指定文件

22、adb shell pm list packages —- 查看手机上面的所有安装的包名

23、adb -s HT114RX11535 logcat —- 当有多台手机设备链接电脑时，adb链接特定的设备

24、adb shell date -s “20131105.093900” —- 修改系统时间的命令

25、（删除system/app下的该apk）
adb shell ls /system/app/MediaWork* (找到system/app下ott apk 的名称，可能每个系统不一致)

adb remount
adb rm /system/app/MediaworkStbClientOTT.apk
adb uninstall com.huawei.mediawork.ott
adb install ….

26、(删除文件 -r, -R, –recursive 指示rm将参数中列出的全部目录和子目录均递归地删除。)
rm -r xxx.xx

27、sqlite命令
在adb shell 中，以sqlite3 tablename来进入数据库模式
.mode [模式名称]来进入相应的模式，一共有八种模式
如：.mode column，.mode line等 我常用.mode column
.header on/off 用来打开或关闭表头
.table 查看数据库中有哪些表

28、adb shell cat -— 查看文件内容

29、查询文件 find命令
进入某一目录，然后使用busybox find xxx*即可

30、##2846579## 打开手机Debug日志

31、位：bit int 4个字节(byte) 32位(bit)

字节：byte

1 byte = 8 bit

1 kb = 1024 byte

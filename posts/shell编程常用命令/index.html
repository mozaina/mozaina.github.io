shell编程常用命令

2020年9月2日
| 阅读

1、diff通常用来比较文件内容，patch通常用来打补丁

diff file1 file2        #以第二个文件为基准,1怎么改就成2了

2c2                     #第一个文件第二行和第二个文件第二行改变了


、< hahaha           #

、> westos           #>第二个文件内容



diff -u file2 file1 >file2.path
patch file2 file2.path      #打补丁后file2就会变得和file1一样

patch -b file2 file2.path   #保留源文件，会多产生一个.orig文件和file1内容相同

diff 比较目录 -r 比较结果为目录中文件的不同，而不是文件内容的不同。

2、cut命令

cut -d : -f 1 passwd        #-d指定截取的分隔符，-f指定截取列
cut -d : -f 1,3 passwd      #第一列和第三列
cut -c 1-3 passwd       #截取1-3字符
cut -c 1，3 passwd       #截取1和3字符的列

3、sort-uniq命令

sort file3          #每行第一个单个数字进行排序
sort -n file3           #正向大小排序
sort -rn file3          #反向大小排序
sort -rn file3 -o num       #指定输出文件
sort -u file3           #去掉重复排序
sort -n file3 | uniq -c     #显示重复次数
sort -n file3 | uniq -d     #只显示重复的行
sort -n file3 | uniq -u     #显示不重复的行
sort -n -k 2 -t ':' test    #按指定分隔符，行排序





4、seq用来输出序列化东西
指定分隔符

seq 100                 #自动生成1-100序列
seq -s '符号#' 100        #选择分割符号
seq -w 100              #自动补位
seq 起始 间隔 结尾
-f                      #使用printf 样式浮点格式
jion jion1 jion2        #第一列相同拼接一起



5、tr命令，转化转换命令

echo "hello westos" | tr "a-z" "A-Z"    #大小写转换
echo "12345" | tr "0-9" "987654321"     #0-9换成9-0，加密解密，可以自己定义加密方案
echo "hello 123westos" | tr -d "0-9"    #删除0-9数字
tr -s " "                               #每个单词用空格表示

6、xargs命令
组合过滤器，也是组合多个命令的工具，可以将管道或标准输入数据转换成命令行参数，也能够从文件中读取数据，也可以将内容转换为其他格式。默认包括换行和空格，不过可以通过命令去除
eg:

find /sbin/ -perm /0700 | ls -l         #管道符无法通过
find /sbin/ -perm /0700 | xargs ls -l       #可以通过

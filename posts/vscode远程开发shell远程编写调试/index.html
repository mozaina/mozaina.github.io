VScode 远程开发 shell 远程编写调试

  • 1，配置免密钥
  • 2，Remote Development插件安装
  • 3，安装格式化、语法工具

1，配置免密钥

  • macOS

  • 查看公钥

    cat /Users/y50/.ssh/id_rsa.pub

  • 没有公钥创建公钥

    ssh-keygen -t rsa

  • macOS公钥导入到服务器

    ssh-copy-id -i /Users/y50/.ssh/id_rsa.pub root@192.168.0.196

  • 指定ssh端口-p 2222

    ssh-copy-id -i /Users/y50/.ssh/id_rsa.pub -p 2222 root@192.168.0.197

  • 免密钥登陆成功

    ssh root@192.168.0.196

2，Remote Development插件安装

  • cmd+shift+p 搜索

  • Remote-SSH: Add New SSH Host

  • 选择配置文件

  • 连接服务器

  • 登陆成功

  • 配置免密钥

    Host 192.168.0.197 HostName 192.168.0.197 User root ForwardAgent yes IdentityFile /Users/y50/.ssh/id_rsa

  • 指定ssh端口

    Host 192.168.0.198 HostName 192.168.0.198 User root Port 2222 ForwardAgent yes IdentityFile /Users/y50/.ssh/id_rsa

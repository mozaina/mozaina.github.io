Linuxshell命令笔记三vi和文件操作

2020年9月2日
| 阅读

vi 的三种功能模式

命令模式

Esc键：进入命令模式

x键：删除当前光标所在的文字

双击d键：删除光标所在整行

h、j、k、l对应方向键左、下、上、右

shift+“+“向下移动一行

n 5 空格 ：光标当前行向后移动5字符

n 5 回车：光标向下移动5行

0：光标快速移动行首

$: 光标快速移动到行尾

H：光标快速移动到第一行第一个字符

10 G：光标快速移动到第10行

gg：光标快速移动到第一行第一个字符

/要搜索的内容：搜索功能，n向下查找，N向上查找

输入模式

i:进入输入模式

O：在上一行进行输入

o：在下一行进行输入

底线命令模式

:：进入底线命令模式

wq! ：w为保存，q为退出，！为强制操作

set nu ：加行号

set nonu ：取消行号

find . -name "文件名" #查找当前目录下的文件 

find /home  -name "*.txt" #查找home目录下的txt文件 

linux支持两种压缩包 tar和zip

zip -v test.zip a.txt #压缩a.txt文件到test.zip里 v为输入信息

zip -r test.zip a.txt test #压缩目录test文件到test.zip里 r为递归

unzip test.zip #解压文件

tar -czvf test.tar.gz test #压缩目录test文件到test.tar.gz里  c为创建

tar -xzvf test tar.gz #解压test.tar.gz x为解压

yum install mysql mysql-server mysql-devel 
#安装mysql，centos自带的命令yum 可以安装多个软件以空格隔开并自动解决依赖   

yum list #列出当前安装的包

service iptables stop #关闭防火墙

chkconnfig iptables off #永久关闭防火墙

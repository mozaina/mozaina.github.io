如何让shell访问正在运行的Docker容器

2020年9月2日
| 阅读

本篇文章将给大家介绍关于如何让shell访问正在运行的Docker容器？下面我们来看具体的内容。

1、使用Docker Attach

可以使用attach命令在Docker容器中访问bash shell。但是你的docker容器必须以/bin/bash启动。

使用以下语法获取Docker容器的shell访问。

$ sudo docker attach

例如，你的docker容器正在运行，其ID为76DEBAD837D2，名称为happy_admin。对attach方法使用以下命令之一。

使用容器ID:

$ sudo docker attach 76debad837d2

使用容器名称：

$ sudo docker attach happy_admin

2、使用Docker Exec

如果docker容器不是用/bin/bash命令启动的。那么就不能使用attach命令。现在，需要使用exec命令在容器中创建bash。确保使用的是Docker版本为1.3之上。

使用以下语法获取Docker容器的shell访问权限。

$ sudo docker exec -it  bash

例如，你的docker容器正在运行，其ID为76DEBAD837D2，名称为happy_tecadmin。将以下命令之一与exec方法一起使用。

使用容器ID：

$ sudo docker exec -it 76debad837d2  bash

使用容器名称：

$ sudo docker exec -it happy_tecadmin bash

以上就是如何让shell访问正在运行的Docker容器的详细内容

shell命令用户管理

2020年9月2日
| 阅读

1. 查看保存用户相关信息的文件 【 cat /etc/passwd 】

【 linux : x : 1000 : 1000 : linux,,, : /home/linux : /bin/bash 】 使用冒号分割

用户名 密码 用户ID 组ID 基本信息 家目录 shell类型

2. 查看保存组的相关信息 【 cat /etc/group 】

【 linux : x : 1000: 】

组名 密码 ID

3. 添加一个用户 【 sudo adduser username 】

4.修改密码

（1） 直接 【passwd】修改当前用户的密码

（2） 修改指定用户的密码 【 passwd username 】

5. 删除一个用户

（1）只删除用户，不删除该用户的工作目录 【sudo deluser username】

（2） 删除用户的同时把工作目录也删除 【deluser –remove-home username】

6. 添加一个用户组 【addgroup groupname】 在 【 /etc/group 】 查看组的相关信息

7. 删除一个用户组 【delgroup groupname】

8. 修改用户属性 【usermod】

常用Shell命令

1、电源管理命令：
poweroff: 关机
reboot：重启
shutdown：关机或重启 （shutdown -h 0 、shutdown -r 0）
//有些命令是可以带参数的，命令和参数、参数和参数之间用一个或多个空格分隔

//有些命令是需要Root权限才能执行的，普通用户没有权限执行

su： select user //切换当前用户

man：manual //查看命令或函数的帮助手册，在Linux世界中，有问题找 “男人”，按Q键退出
//Ctrl+C组合键：强制终止当前正在执行的命令

2、网络配置管理命令
ifconfig：查看或设置IP地址等网络配置
ping：检测本机与目标机器之间的网络是否通畅
netstat：查看当前网络状态信息

clear：清屏
reset：重置终端

3、进程（任务）管理命令
进程（Process）：正在运行的程序，或者程序的一次执行过程，一个进程就相当于一个任务
每个进程都有一个不同的编号，即PID，操作系统通过PID来区分不同的进程

Linux等现代操作系统都是多任务的系统

ps：process snapshot //查看进程快照信息，常用参数 -e，-aux
top：查看进程的实时信息（Q键退出）
kill：kill -9 “PID” 结束单个进程，不同进程 PID 不同
killall： killall “进程名” 批量结束进程

在Linux系统中，一个进程通常都对应一个控制终端，不依附于任何终端的进程就是守护进程

&：将命令调到后台执行
fg：foreground，将当前在后台的进程调到前台执行
bg：background，运行处于后台的所有进程
Ctrl+Z组合键：强制将当前在前台执行的进程调到后台，并暂停进程的执行

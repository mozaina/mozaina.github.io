PHP Web Shell v1.0.1 说明文件

一、程序说明：

1.程序名称：PHP Web Shell
2.程序版本：v1.0.1
3.开发平台：WindowsXP professional + PHP 5.0.2
4.运行平台：Win32/Linux/Unix/BSD + PHP 4.3 以上等多种系统平台

二、功能说明：

1.文件/目录的各种编辑管理
2.执行系统命令
3.执行SQL查询
4.查看系统信息和PHP环境信息
5.测试服务器性能
6.修改登陆用户名和密码

注意事项：
请在把程序上传到服务器之前修改程序的配置资料，特别要填正确本程序的正确的文件名，
另外，请不要直接使用本程序的编辑功能来编辑本文件，否则将出现致命错误，切记！

三、更新历史

2004-02-18 v0.4 原始版本，只具有执行命令功能
2004-04-20 v0.5 修改版本，增加了文件编辑等很多功能
2004-12-15 v1.0.0 功能整和成一个文件，容易传播使用
2004-12-23 v1.0.1 发现编辑程序本身导致致命的bug，增加修改登陆信息功能

Secureshell树莓派远程控制

2020年9月2日
| 阅读

关于不使用显示器远程登陆树莓派系统方法

1、通过远程SSH使用树莓派

(1)、打开SSH（Secure Shell）使Linux的一个可以主机电脑上快速打开和使用树莓派的一个终端会话。

在树莓派中首先要打开terminal终端命令行打开SSH支持，

命令：sudo raspi-config（打开树莓派设置界面）

选择Advanced Options选项，回车

显示SSH server已经可以用

2、在window需要下载一个叫做”putty”的免费软件s上通过SSH连接树莓派,

首先需要在windows上安装一个叫Putty的东西，打开它，输入树莓派的IP地址，端口号，点击Open

然后依次输入用户名和密码（一般用户名为Pi，密码raspberry）

3 可以通过如下的命令查看树莓派已经识别的wifi

命令如下：sudo iwlist wlan0 scan

其中ESSID是我们扫描到的无线名称，

4、树莓派中安装软件

升级软件包，

命令：sudo apt-get update

更新软件包命令：sudo apt-get upgrade,

安装软件包：

命令如下：sudo apt-get instal +软件包

卸载软件包：

命令如下：sudo apt-get remove +软件包

卸载旧的依赖

但你安装一个包，你可以注意到几个其他的包也被安装了，你就意味这个包需要其他的包来执行。这就叫依赖。但你移除软件包，这些旧的依赖包会留在系统。它们可以安全地删除通过如下命令：

命令如下：sudo apt-get autoremove

5、怎么用VNC远程控制树莓派

安装VNC server

sudo apt-get install tightvncserver

首先在树莓派的命令行界面输入

sudo apt-getinstall tightvncserver

回车

等待安装完毕

安装完毕之后，输入tightvncserver回车，启动vnc提示输入密码

输入两个密码，一个控制桌面，一个view only,我一般都设置成一样，比如******。然后就会在当下目录生成一个.vnc的隐藏目录。ps.如果想重置密码，只需rm .vnc -r 然后再次输入tightvncserver即可。

设置完密码之后，树莓派需要运行vnc服务

再次输入tightvncserver

回车如下图

打开客户端的VNC Viewer

打开http://www.tightvnc.com/download.php下载windows需要安装的VNC-Viewer 解压之后一路next安装，最后运行程序如下图，把192.168.1.222（这是树莓派在我局域网的ip，具体可以通过路由器查看，或者设置成固定值），换成你的树莓派的ip。点击connect，输入之前设置的密码，点击ok。

输入刚刚设置的密码，raspberry

6、树莓派上安装python

(1) sudoapt-get install python-dev

(2) 更新：sudo easy_install –U distribute

(3) 安装python-pip: sudo apt-get install python-pip

(4) 安装python的GPIO库： sudopip install rpi.gpio

Windows10关于Ruby安装笔记

2020年9月2日
| 阅读

官网下载实在太慢，码住这位聪明网友的指导成功上岸！

1.下载ruby
下载地址：https://rubyinstaller.org/downloads/

如果觉得下载速度过慢，可以在介为朋友的的百度云下载：

链接：https://pan.baidu.com/s/1i6NjnfAM3pHB-bbbBjp9Fw
提取码：to70

2.就直接按它的默认配置，安装即可
安装成功之后，输入 ruby -v可获得你安装的版本号。

3.更换gem源
我们使用淘宝的gem源https://gems.ruby-china.com/。这些命令都是在 CMD 里面执行的。

gem sources –add https://gems.ruby-china.com/ –remove https://rubygems.org/

执行结果如下图：

4.检测是否替换成功
gem sources -l

注意这是小写字母 l 不是数字 1。

正确的返回结果如下：

5.安装sass
gem install sass
安装成功返回如下图：

6.安装compass
gem install compass
安装成功返回如下图：

7.可以通过 sass -v 和 compass -v 查看安装的版本号

到这里，就说明我们已经成功啦！

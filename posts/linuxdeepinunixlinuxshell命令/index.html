2020年9月2日
| 阅读

终端操作
在LinuxDeepin中，您可以通过主菜单中的“附件——终端”进入图形化终端使用。使用时，输入命令并按回车键执行。当您需要输入密码时，屏幕上既不会显示您的密码也不会显示*等提示符。当您结束在终端的工作时，可以按Ctrl+D关闭终端。

您也可以按Ctrl+Alt+F1进入文本终端，再按Alt+F7回到图形界面。（按Ctrl+Alt+Fx进入ttyx，其中tty1至tty6是文本终端，tty7至tty12是图形界面，默认的图形界面开在tty7） 结束在一个文本终端的工作后您可以按Ctrl+D登出。

Ctrl+C 结束当前命令
Ctrl+Z 挂起当前命令。以fg或者bg命令恢复
Ctrl+D 登出当前会话，类似于输入”exit“并回车
Ctrl+W 删除前一个单词（以空格判定）
Ctrl+U 删除本行在光标前的所有内容
小键盘上下键 切换最近输入过的命令
！！ 上一条命令。例如：以普通用户执行apt-get update会提示权限不足，此时输入sudo !!即可
Ctrl+A 回到行首
Ctrl+E 前进到行尾
Alt+B 光标回移一个单词
Alt+F 光标前进一个单词。这个操作会于终端快捷键冲突
Ctrl+Shift+T 开启一个新的标签页，类似于浏览器中的Ctrl+T
Ctrl+L 清屏。实际上是把屏幕上已有的字向上移动
Ctrl+J 回车。如果键盘回车键没有正确映射，Ctrl+J可以用上
查看帮助文档
man COMMAND 查看COMMAND命令的帮助。Linux下所有的软件都有简明的man文档，在遇到不会用的软件或者不会用的功能的时候，最好先仔细查看man文档，如果弄不懂再去问人

文件操作
ls 显示当前目录下的文件和文件夹
ls -la 显示当前目录下所有文件和文件夹，包括隐藏文件，并显示文件详细信息
cd DIR 切换到DIR文件夹下。此处的DIR可以是绝对路径，如”/home/sun/downloads”，也可以是相对路径，如”./downloads”
pwd 查看当前的工作目录。默认的工作目录在~下。~是您的“家目录”，默认在/home/USERNAME
mkdir DIR 在当前目录下创建“DIR”文件夹
rmdir DIR 移除空文件夹DIR
rm FILE 删除当前目录下的“FILE”文件
rm -r DIR 递归删除文件夹DIR，也就是删除DIR及其中的所有文件、文件夹。
cp FILE1 FILE2 将FILE1复制到FILE2，包含了复制和重命名两个结果。例如：
cp test.text test2.text #将test1.text复制到同一目录下，并命名为test2.text
cp test.text /home/sun/ #将test.text复制到/home/sun文件夹下，保留原有文件名
mv FILE1 FILE2 移动/重命名文件。规则与“cp”相同。
ls -s FILE LINK 创建文件LINK作为FILE的软链接。软链接只是一种快捷方式，当原文件改变地址时会失效
touch FILE 若FILE文件不存在，此命令创建空文件FILE；若FILE文件已存在，则更新“访问时间“
more FILE 显示文件内容。如果文件不是文本文档则不会显示内容，此时可以使用”bvi FILE“进行十六进制显示
head FILE 显示文件的前10行
tail FILE 显示文件的末10行
tail -f FILE 动态地显示文件的末10行。如果文件改变，则显示内容也即时改变。
搜索
grep PATTERN FILES
在文件中搜寻特定片段

grep -r PATTERN DIR
在文件夹中递归地寻找文件中的特定片段

COMMAND | grep PATTERN
在COMMAND的输出中寻找特定片段。例如：lspci | grep VGA #在lspci的结果中搜寻带有VGA字样的行。lspci是查看计算机pci硬件的命令，其中带有VGA字样的应该是显卡信息

locate FILE 寻找特定名字的文件。locate命令是从数据库中查询文件，因此很快，但是数据库每24小时才会生成一次，可能无法找到特别新的文件。
find DIR -name FILENAME
在指定的文件夹（不指定则为当前工作目录）下寻找特定名称的文件。例如：

find / -name xorg.conf #在系统的所有目录中寻找叫做”xorg.conf“的文件或文件夹
find /etc -name xorg.conf* #在/etc下的所有目录中寻找以”xorg.conf“开头的文件或文件夹
由于一些系统文件夹的权限设置，搜索的过程中会有一些”Permission Denied“的报错，一般情况不会影响，因为大部分系统配置文件都是普通用户可读的。

进程管理
ps 显示当前活跃进程
ps aux 显示所有进程
top 动态显示所有进程
kill PID 结束某个进程。注意如果进程的用户是root，只有用root或者sudo才能结束
killall PROCESS 结束某个进程。PROCESS处填写进程名
bg 列出挂起或后台运行的进程；在后台恢复一个挂起的进程
fg 把最近的在后台开启的进程拉至前台
fg N 把N号进程拉至前台

系统信息
date 显示当前日期、时间。
在LinuxDeepin上与网络校准时间的方式是安装ntpdate

sudo apt-get install ntpdate
sudo ntpdate time.ustc.edu.cn #此处填写一个Time-Server的地址
cal 显示日历。例如：
cal #显示当月日历

uptime 显示机器已运行时间
w 显示当前计算机的已登录用户和uptime等信息
who 显示已登录用户
who -u 显示已登录用户并显示登录进程（在最右边）。利用这个命令看到登录进程之后可以踢出登录用户。
uname -a 显示操作系统内核信息
more /proc/cpuinfo 显示CPU信息
more /proc/meminfo 显示内存信息
lspci 显示PCI硬件信息
df 显示磁盘使用信息。默认单位是字节。可以使用df -h方便阅读
du 显示文件夹磁盘占用量。du命令的选项较多，可以man du看以下
free 显示内存和虚拟内存使用情况。使用”free -m“以MB为单位显示
whereis BIN 显示BIN文件的位置。只会显示在PATH里的可执行文件，例如：
whereis whereis #显示whereis的位置
Linux下的命令大多是PATH里的一个可执行文件，可以是编译型语言的二进制码，也可以是解释型语言的脚本
which APP 显示哪个软件会被默认执行

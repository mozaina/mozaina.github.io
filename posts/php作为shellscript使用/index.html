PHP作为ShellScript使用

2020年9月2日
| 阅读

Window环境下运行PHP脚本

**
**

主要有2种方式在Win环境下运行PHP脚本

1、直接在类似PHPStorm的IDE中运行，使用IDE的内置解释器运行；

2、调用php解释器，定位到php/bin目录下，命令行中运行

  1. php C:\user\desktop\test.php

可以将PHP的bin路径添加到环境变量的Path路径中，这样可以在任意位置调用php就解释器；

①【path】变量添加php类库所在的路径，如： C:\php5.5

②新建系统环境变量【PHPRC】，值为 php.ini 所在的目录，如： C:\php5.5

  1. 可以使用以下命令验证php是否添加入环境变量
  2. php -v

※设置cmder的环境变量【setting >> Startup >> Environment】set PATH=PHP bin 路径

Linux环境下本地运行PHP脚本

**
**

1、使用PHP解释器运行

①将php的bin目录添加到系统变量

编辑 ~/.bash_profile 文件，添加：

  1. PATH=$PATH:$HOME/bin:php的bin路径
  2. 如：
  3. PATH=$PATH:$HOME/bin:/usr/local/php/bin

使PHP环境生效

  1. . ~/.bash_profile

②使用php解释器运行

  1. php test.php

注：通过这种方法，不用在php脚本头添加解释器路径说明；

2、 在编写PHP脚本时，在文件首添加解释器路径说明：

  1. 【test.php】
  2. **#!/usr/local/bin/php -q**
  3. ``
  4. echo 'Hello world!';
  5. ?>

运行脚本（将脚本添加运行权限）

  1. ./test.php

注： 路径说明中的 -q 参数是为了抑制header的输出；

PHP脚本获取控制台参数

**
**

有两个获取参数的特殊变量

$argc：记录参数数量

$argv：记录参数的数组

示例：

  1. **【 test.php】**
  2. ``
  3. for($i=0;$i<$argc;$i++){
  4. echo $argv[$i];
  5. }
  6. ?>
  7. ``
  8. ``
  9. **【控制台调用】**
  10. php test.php a b c d e f


PHP调用本地指令

1、exec 函数

原型: string exec ( string $command [, array & $output [, int & $return_var ]] )

返回执行结果的最后一行；

参数： $command ：本地指令的字符串

$output：本地指令执行后的输出结果，将其装载在数组中，如果$output变量要重复使用，最好每次输出后清空该数组 unset（$output）；

$return_var：获取执行的状态码，执行成功返回 0；

示例：

  1. ``
  2. exec("ls -l",$output,$return_var);
  3. if($retrun_var){
  4. echo $output;
  5. }
  6. ?>

2、system 函数

``

原型: string system ( string $command [, int & $return_var ] )

system函数直接将结果输出到浏览器中，其他参数同exec

示例：

  1. ``
  2. system("ls");
  3. ?>

3、shell_exec 函数

原型: string shell_exec ( string $cmd )

返回执行指令的结果；

可以使用 代替shell_exec函数

示例：

  1. ``
  2. $result = shell_exec('ls');
  3. echo $result;
  4. 等同：
  5. $result =ls;
  6. ?>

PHP执行本地指令超时

**
**

使用PHP执行本地指令时，由于php.ini 中 ** max_execution_time** 一般默认为30s，所以在执行一些耗时指令时可能回导致超时，解决方法有以下两种：

1、取消运行时间限制

在脚本的开头添加：

  1. ini_set('max_execution_time',0); //取消运行时间限制
  2. ini_set('memory_limit',0); //取消运行内存限制

2、将该指令执行的线程结果进行重定向；

  1. php
  2. system("cp -r /a /b > /tmp/temp_file ");
  3. ?>

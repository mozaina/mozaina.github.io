test条件判断

  1. 按照文件类型进行判断
    判断语句1 .test 选项 文件
    test -d /root 判断/root文件是否存在 并且是否是目录
    test -e sofency.txt 判断文件是否存在
    test -f sofency.txt 判断文件是否存在，并且文件是否是普通文件
    判断语句2 .[空格选项 文件空格]
    [ -e /root ] true

  2. 按照文件权限进行判断

    形式：[ -r 文件 ]

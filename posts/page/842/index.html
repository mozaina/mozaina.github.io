shell比较字符串是否相等

2020年9月2日
比较两个字符串是否相等的办法是： if [ “$test”x = “test”x ]; then 这里的关键有几点： 1 使用单个等号 2 注意到等号两边各有一个空格：这是unix shell的要求 3 注意到”$test”x最后的x，这是特意安排的，因为当$test为空的时候，上面的表达式就变成了x = testx，显然是不相等的。而如果没有这个x，表达式就会报错：[: =: unary operator expected 二元比较操作符,比较变量或者比较数字.注意数字与字符串的区别. 整数比较 -eq 等于,如:if [ “$a” -eq “$b” ] -ne 不等于,如:if [ “$a” -ne “$b” ] -gt 大于,如:if [ “$a” -gt “$b” ] -ge 大于等于,如:if [ “$a” -ge “$b” ] -lt 小于,如:if [ “$a” -lt “$b” ] -le 小于等于,如:if [ “$a” -le “$b” ] <= 小于等于(需要双括号),如:((“$a” <= “$b”))……

Linux一些常用shell命令

2020年9月2日
| 阅读

Linux常用命令

mac登录到linux

ssh username@hostname

查看磁盘大小:

df -h

查看当前文件夹的大小

du -sh

查看当前文件夹下的所有文件的大小

du -hd 1 ./*

文件下载

scp root@服务器IP:/mnt/nohup.out /Users/devnn/Desktop/
将服务器上/mnt/nohup.out文件下载到桌面

文件上传

scp /Users/devnn/Desktop/abc.txt root@服务器IP:/mnt
将桌面abc.txt文件上传到服务器/mnt目录下

adb常用命令

android查看签名文件md5信息(默认口令android)

keytool -v -list -keystore 密钥文件路径

android给apk签名

jarsigner -verbose -keystore xxx.jks(签名文件) -signedjar xxx.apk（签名后的apk） xxx.apk（需要签名的apk） xxx（keystore别名)

android进入到手机shell命令行模式

adb shell
(可以像进入linux系统那样查看手机文件系统)

杀死adb进程

adb kill-server

开启adb进程

adb start-server
(开发时,设备或模拟器掉线的时候,这两个命令很管用)

adb命令安装apk

安装apk
adb install apk路径
缷裁apk
adb uninstall 包名

adb命令从手机复制文件到电脑

adb pull /data/data/xxx/xxx.txt /Users/devnn/xxx

adb命令从电脑复制文件到手机

adb push /User/xxx/xxx.txt /data/data/xxx

adb执行shell命令

adb shell 【命令内容】
例如：
adb shell su chmod 777 /data/data/com.kuaiditu.user/databases/kuaiditu_user.db

adb查看最上层activity名称

linux:
adb shell dumpsys activity | grep “mFocusedActivity”

windows:
adb shell dumpsys activity | findstr “mFocusedActivity”

CentOS7安装MySQL两种安装方式在shell中输入相关命令安装

2020年9月2日
| 阅读

安装MySQL一般来说有两种安装方式：

第一种是直接在shell中输入相关命令进行下载。具体的输入命令流程简要概述如下：

  1. 输入命令：wget https://dev.mysql.com/get/mysql80-community-release-el7-2.noarch.rpm
    使用wget直接在路径https://dev.mysql.com/get/mysql80-community-release- el7-2.noarch.rpm中下载 ：
    mysql80-community-release-el7-2.noarch.rpm 文件。

  2. 输入yum命令安装刚才下载好的rpm安装包：
    yum -y install mysql80-community-release-el7-2.noarch.rpm

  3. 输入yum命令：
    yum -y install mysql-community-server
    安装MySQL的社区版服务器。

  4. 执行上述命令后，CentOS需要运行一段时间，安装完成后会覆盖CentOS默认消减版MySQL：mariadb。

  5. 输入命令：systemctl start mysqld.service
    启动新安装完成的MySQL。开始进行MySQL的初始化以及账户密码等初次配置。

  6. 输入命令：systemctl status mysqld.service
    查看用于控制MySQL数据库系统的程序 mysqld 服务的运行状态。注意观察各段字符串中的Active字符串的后面的高亮字符串是字符串active (running)还是 inactive (dead)。如果是active (running)说明MySQL服务已经启动，如果是inactive (dead) 说明MySQL服务已经关闭。

  7. 输入命令：grep “password”/var/log/mysqld.log
    获取保存在error log文件中MySQL8为root用户随机生成了一个初始随机密码。该密码位于字符串：
    A temporary password is generated for root@localhost:
    冒号的后面。
    使用者需要记住这个临时root密码，在初次登录MySQL时需要输入此密码，然后才能进入MySQL修改密码。

  8. 输入命令：mysql -u root -p
    初次登录MySQL。根据提示输入之前记下的临时密码。

  9. 在myswql> 标头下输入命令：
    mysql> ALTER USER ‘root’@ ‘localhost’ IDENTIFIED BY ‘NewPassword’;
    注意将上述命令这种的’NewPassword’字符串改成使用者的root账号的密码。

  10. 输入命令：mysql>flush privileges;
    使用上述命令，刷新MySQL的系统权限配置表。使得上述所有的配置生效。或者直接输入重启MySQL命令： service mysqld restart ：使得上述所有配置生效。

  11. 完成配置,输入命令：
    mysql>exit; ：或者
    mysql>quit; ：或者
    mysql> \q;
    退出MySQL编辑模式。

  12. 至此，采用第一种安装方式安装MySQL全部完成。

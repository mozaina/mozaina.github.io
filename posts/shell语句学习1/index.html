shell语句学习1

2020年9月2日
| 阅读

Linux中的shell类似windows下的cmd.exe。shell是系统的用户界面，提供了用户与内核进行交互操作的接口，shell介于操作系统和用户之间，它是用户与内核的翻译官，是一个命令解释器。

shell也有不同的版本:

Bourne Shell：是贝尔实验室开发的。
Bash：是GNU的Bourne Again Shell，是GNU操作系统上默认的shell。
Korn Shell：是对Bourne SHell的发展，在大部分内容上与Bourne Shell兼容。
C Shell：是SUN公司Shell的BSD版本。
Z Shell：The last shell you’ll ever need! Z是最后一个字母，也就是终极Shell。它集成了bash、ksh的重要特性，同时又增加了自己独有的特性。

Shell 脚本（Shell Script），是一种为Shell编写的脚本程序。业界所说的Shell通常都是指shell脚本，但读者朋友要知道，Shell和Shell Script是两个不同的概念。由于习惯的原因，简洁起见，本书中出现的”Shell编程”都是指Shell脚本编程，不是指开发Shell自身。
Shell 编程跟java、php编程一样，只要有一个能编写代码的文本编辑器和一个能解释执行的脚本解释器就可以了。本书关注的是 Bash，也就是 Bourne Again Shell，由于易用和免费，Bash在日常工作中被广泛使用。同时，Bash也是大多数Linux系统默认的Shell。在一般情况下，人们并不区分 Bourne Shell 和 Bourne Again Shell，所以，像 #!/bin/sh，它同样也可以改为#!/bin/bash。
打开文本编辑器(可以使用vi/vim命令来创建文件)，新建一个文件test.sh，扩展名为sh（sh代表shell），扩展名并不影响脚本执行，见名知意就好，如果你用php写Shell 脚本，扩展名就用php好了。
输入一些代码，第一行一般是：#!/bin/bash
示例：文件名称：hello_world.sh

#!/bin/bash
echo “hello world !”

执行结果如下：

[root@ ]# ./hello_world.sh
hello world !

“#!” 是一个约定的标记，它告诉系统这个脚本需要什么解释器来执行，即使用哪一种Shell。
echo命令用于向窗口输出文本。

运行shell脚本的方法有两种：

1、作为可执行程序

[root@ ]# mkdir /shell   #创建shell文件夹
[root@ ]# vi hello_world.sh  #创建hello_world.sh文件并写入脚本
[root@ ]# chmod +x hello_world.sh #使脚本具有执行权限（在root下设置）
[root@ ]# ./hello_world.sh #执行脚本
hello world !

2、作为解释器参数
这种运行方式是，直接运行解释器，其参数就是Shell脚本的文件名
如：

[root@ ]# /bin/sh hello_world.sh
[root@ ]# /bin/php hello_world.php

这种方式运行的脚本，不需要在第一行指定解释器信息，写了也没用。

脚本示例：打印有色彩的文本

[root@ shell]# vi print_text_colorful.sh
#!/bin/bash
#清空屏幕显示
clear # 加粗效果
echo -e "\033[1m Hello World" # 闪烁效果
echo -e "\033[5m Blink" # 回归正常
echo -e "\033[0m Hello World" # 红色
echo -e "\033[31m Hello World" # 绿色
echo -e "\033[32m Hello World" # 黄色
echo -e "\033[33m Hello World" # 查看显示效果
echo -e "\033[34m Hello World"
echo -e "\033[35m Hello World"
echo -e "\033[36m Hello World" # 回归正常
echo -e -n "\033[0m"
echo -e "\033[41m Hello World"
echo -e "\033[42m Hello World"
echo -e "\033[43m Hello World"
echo -e "\033[44m Hello World"
echo -e "\033[45m Hello World"
echo -e "\033[46m Hello World"
echo -e "\033[0m Hello World"

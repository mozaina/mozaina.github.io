Hbase shell删除表失败，提示信息
hbase(main):047:0> drop ‘user’

ERROR: Table user is enabled. Disable it first.

Drop the named table. Table must first be disabled:
hbase> drop ‘t1’
hbase> drop ‘ns1:t1’

ERROR: Table user is enabled. Disable it first.
表示用户已启用，先禁用它
删除表只能在表状态为：disable 状态下删除
如何检查表是否启用或禁用：
is_enabled ‘表名’
is_disabled ‘表名’

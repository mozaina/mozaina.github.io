立即关机与重启

在命令行键入以下命令，系统立即关机：

linux@ubuntu:~$ sudo shutdown -h now

linux@ubuntu:~$ sudo poweroff -h now

在命令行键入以下命令，系统立即重新启动：

linux@ubuntu:~$ sudo shutdown -r now

linux@ubuntu:~$ sudo reboot now

定时关机与重启：

linux@ubuntu:~$ sudo shutdown -h +45 "That is all,game over."

linux@ubuntu:~$ sudo shutdown -r +60

第一条命令可用于某一时间段后需要定时关机的场景，比如设置考试时限。后面的”That is all,game over.“是提示。

第二条命令则是重启，60分钟后重启。可以用于“系统正在安装软件，安装大概需要40分钟，安装完毕需要重启电脑”这一场景。用户不需在一旁等候。

三条命令在请求输入密码后将会被执行。此处为了演示，按下ctrl+c终止了命令。

shell命令格式

username@hostname:direction$

username：用户名，显示当前登录用户的账户名

hostname：主机名，显示登录的主机名，例如，远程登录后显示的主机名。

direction：目录名，显示当前所处路径，当在根目录下显示为“/”，当在用户主目录下，显示为“~”

$Command [-Options] Argument1 Argument2 ...

$：Shell提示符，表示其他用户。如果是超级用户，用”#“表示。

Command：命令名称，Shell命令或程序，严格区分大小写。

Options：命令选项，用于改变命令执行动作的类型，由“-”引导，有的由“–”引导。可以同时带有多个选项。

Argument：命令参数，指出命令作用的对象或目标，有的命令允许带多个参数。

通常一条命令包含三个要素：命令名称、选项、参数。命令名称必须有，选项和参数可有可无。

一条命令的三要素之间用空格隔开，sudo shutdown -r 三者之间空格隔开。

linux@ubuntu:~$ cd /; ls; date

若将多个命令在一行书写，用分号（;）将命令隔开。

注：在一行当中键入多个命令，其间效果是有影响的。

linux@ubuntu:/$ ls /etc/ \
> network -l

如果一条命令不能在一行写完，在行尾使用反斜杠（\）标明该命令未结束。

命令行操作

bash除了在命令编辑功能上比sh有很大改进外，还增加了特有功能，极大地方面了用户在Shell命令行上的操作。

补齐命令与文件：

在使用Shell命令时，很多用户经常会遇到命令或文件没有记全的情况。Bash Shell的命令和文件名补齐功能会帮助用户，在束缚命令或文件名的几个字符后，按TAB键或ESC键自动补齐剩余没有输入的字符串。

查询历史命令

history[numberline]

显示history只能记录有限条的历史命令，默认保留500条命令。

Bash Shell将历史命令容量保存在环境变量HISTSIZE中。

linux@ubuntu:~$ echo $HISTSIZE

linux@ubuntu:~$ HISTSIZE=1000


linux@ubuntu:~$ history

输入本条命令，随即显示最近的1000条命令。

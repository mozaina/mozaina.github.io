二、循环语句

1、类C循环

#!/bin/bash
for ((i=0; i<=10; i++)) 
do    
echo "hello $i" 
done

2、for ..in循环

#!/bin/bash
for i in {1,2,3,4}
do    
    echo "hello $i"
done

或者：

#!/bin/bash
for i in {1..10} 
do    
    echo "hello $i" 
done

3、while循环

#!/bin/bash
i=0 
while [ $i -le 5 ] 
do  
    echo "hello $i"  
    let i++ 
done

4、until循环

#!/bin/bash
i=0 
until [ $i -ge 10 ]
do  
    echo "hello $i"  
    let i++
done
死循环

1、类C循环：

#!/bin/bash
for (( ; ; ))
do 
     echo "hello bit"
done

2、while循环

#!/bin/bash

while : 
do  
    echo "hello bit" 
done 
或者 
#!/bin/bash
while true 
do  
    echo "hello bit" 
done

3、until循环

#!/bin/bash
until false 
do  
     : #空语句，什么也不做
done

命令行循环

命令行循环就是将写在shell脚本中的语句写到命令行，写法完全一样。

i=0; while [ $i -le 10 ]; do echo "hello bit $i"; let i++; done

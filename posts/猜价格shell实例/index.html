猜价格shell实例

2020年9月2日
| 阅读

猜商品价格的游戏

用户输入的范围为0-999之间

用户每次猜会显示价格高了或低了,并显示当前是猜的第几次

如果猜对了则显示恭喜并退出

1 #!/bin/bash

2 a=expr $RANDOM % 1000

3 b=0

4 echo “price(0-999): “

5 while true #一直循环while

6 do

7 read -p “input price: “ p #接收用户输入

8 let b++

9 if [ $p -eq $a ]

10 then

11 echo “congrantiuation!!!”

12 echo “time is $b”

13 exit 0 #退出

14 elif [ $p -gt $a ]

15 then

16 echo “is too high”

17 echo “time is $b”

18 else

19 echo “is too low”

20 echo “time is $b”

21 fi

22 done

扩展 进行三轮猜价格

1 #!/bin/bash

2 x=1

3 while [ $x -le 3 ] #三轮结束

4 do

5 a=expr $RANDOM % 100

6 b=0

7 echo “num:$x price(0-99): “ #提示第几轮

8 while true

9 do

10 read -p “input price: “ p

11 let b++

12 if [ $p -eq $a ]

13 then

14 echo “congrantiuation!!!”

15 echo “time is $b”

16 break #猜测正确跳出当前循环

17 elif [ $p -gt $a ]

18 then

19 echo “is too high”

20 echo “time is $b”

21 else

22 echo “is too low”

23 echo “time is $b”

24 fi

25 done

26 let x++

27 done

PandoraBox & Shell基本命令

ip
ssh
scp
opkg————-OpenWRT PacKaGe。在pandoraBox中可用来下载安装软件，newifi里的话，可能有用
opkg update
opkg -d usb install *
export
chmod
vi—————Linux中常用的文本编辑器，具体用法请自行Google。如果觉得vi学起来麻烦，也可以用在电脑上编辑好文件再用winSCP传上去。
uhttpd
test
dropbear
cat
cd ————–change directory切换目录，例如，『cd config』切换到当前目录下的config目录，『cd ..』将切换到上级目录，『cd /etc』切换到根目录下的目录『etc』
wget
mtd
dd
ls—————list 。列表显示文件和文件夹。蓝色表示目录，绿色表示可执行文件，白色表示普通文件。『ls -l』显示详细信息。『ls /etc』显示etc文件夹下的内容，留空则显示当前目录中的内容。
iptables
crontab
find————-搜索文件
ll—————等同于『ls -l –color=tty』，简便写法。
grep————-Global Regular ExPression搜索文本，请配合正则表达式服用。
cp—————CoPy复制文件。虽然是基本命令但我好像从来没用过
sed————–似乎是可以用在脚本中的文本编辑器，嘛这个真的没(wo)啥(bu)用(hui)啦～

ssh进后台就是linux的命令行界面

vi /etc/rc.local的意思是使用linux下的编辑器编辑文件/etc/rc.local

该文件是linux开机启动会执行的文件

vi==编辑器，如果你需要再linux命令行下编辑文件，必须学会使用vi编辑器

可供参考资料：http://www.leiphone.com/author/hoowa

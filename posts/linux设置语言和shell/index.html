一，选择默认语言和输入法

点开应用程序（左上角），系统工具，选择设置
进入后点开区域和语言，也就是蓝色旗帜

进入后上面两栏是选择默认语言，下面俩栏添加输入法

二，开启虚拟机和可能的报错

[kiosk@foundation28 Desktop]$ rht-vmctl start desktop
#开启红帽desktop虚拟机

Error: desktop not started (is already running)         #重复输入显示正在运行

[kiosk@foundation28 Desktop]$ rht-vmctl view desktop #显示虚拟机

     Error: unable to view desktop - not currently running.  #虚拟机未开启

[kiosk@foundation28 Desktop]$ rht-vmctl poweroff desktopPowering off desktop. #关闭虚拟机

  desktop和server虚拟机        账户student密码student        管理员root密码redhat

[kiosk@foundation28 Desktop]$ rht-vmctl reset desktopAre you sure you want to reset desktop? (y/n) y #重置虚拟机

三，shell的基础介绍

#命令行字符的含意#####################################

                [kiosk@foundation28 Desktop]$

kiosk #当前用户名称

@ #分隔符

foundation28 #主机名称

Desktop（桌面） #系统所在位置

$ #（用户身份$为普通用户#为超级用户）

参数

一道杠 #单词缩写

两道杠 #单词全拼

-a -b -c=- abc的任意顺序

获得帮助

mandb #更新帮助信息，在whatis ()没结果时用

whatis 命令 #查看命令用途

命令–help #查看命令的帮助

[] #可选加

… #加的个数任意

<> #必须加入

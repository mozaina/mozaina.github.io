SHELL脚本中日期的使用以及时间变量的使用

2020年9月2日
| 阅读

1.SHELL脚本 中date的使用

#（1）最简单的使用

echo “今天日期是: date +%Y%m%d“

今天日期是:20180824

#（2）时间变量的使用

#!/bin/bash

echo “时间是：date '+%Y%m%d %H:%M:%S'“

echo “时间是：date '+%Y-%m-%d %H:%M:%S'“

时间是：20180824 08:11:17


时间是：2018-08-24 08:11:49

#（3）简单的shell脚本

#在当前目录下面创建当天日期的目录

##written by 2018-8-24

#!/bin/bash

guoch=date +%Y%m%d

mkdir ${guoch}

#（4）前一天的日期

#前一天的日期

#（今天是24号）

echo “date -d "-1 day" +%d“

结果是：23

#前一小时

#当前是16点
echo “date -d "-1 hour" +%H“

结果是：15

#前一分钟

echo “date -d "-1 min" +%M“

#前一秒钟

echo “date -d "-1 second" +%S“

#（5）学习日期变量的简单应用

#!/bin/bash
##In this Script we will use variables
##Writen by 2018-08-24

d1=date +%H:%M:%S
echo “The Script begin at $d1”
echo “Now we will sleep 2s”
sleep 2 #休眠2秒
d2=date +%H:%M:%S
echo “The Script end at $d2”

测试结果是：

###上面涉及到sleep的应用

sleep 1 睡眠1秒
sleep 1s 睡眠1秒
sleep 1m 睡眠1分
sleep 1h 睡眠1小时

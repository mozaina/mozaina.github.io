MySQLShell80官方手册第二章安装

2020年9月2日
| 阅读

一、在 Windows 上安装

要使用 MSI 安装程序在 Windows 上安装 MySQL Shell，请执行以下操作：

  1. 从 http://dev.mysql.com/downloads/shell/ 下载 Windows（x86，64 位）MSI 安装程序包；
  2. 出现提示时，单击 Run ；
  3. 请按照安装向导中的步骤进行操作。

重要
在安装 MySQL Shell 之前，请确保已在 Windows 系统上安装了 Visual Studio 2015 的 Visual C ++ 可再发行组件（可从 Microsoft 下载中心获得）。

二、在 Linux 上安装

注意
MySQL Shell 的安装软件包仅可用于有限的 Linux 发行版，并且仅可用于 64 位系统。

在 Linux 上安装 MySQL Shell 最简单方法是使用 MySQL APT Repository 或 MySQL Yum Repository。 对于不使用 MySQL 系统信息库的系统，还可以直接下载并安装 MySQL Shell。

2.1 使用 MySQL APT Repository 安装

对于 MySQL APT Repository 支持的 Linux 发行版，请遵循以下路径之一：

  • MySQL APT Repository 没有作为系统上的软件存储库：

    • 请按照《添加 MySQL APT Repository》中给出的步骤进行操作，尤其要注意以下几点：

      • 在配置包的安装过程中，在对话框中要求您配置存储库时，请确保选择 MySQL 8.0 作为所需的发行系列。

      • 更新软件包信息：

                sudo apt-get update

    • 安装 MySQL Shell：

          sudo apt-get install mysql-shell

  • MySQL APT Repository 已经作为系统上的软件存储库，请执行以下操作：

    • 更新软件包信息

          sudo apt-get update

    • 更新配置：

          sudo apt-get install mysql-apt-config

在对话框中要求您配置存储库时，请确认版本。

* 安装 MySQL Shell：

            sudo apt-get install mysql-shell

2.2 使用 MySQL Yum Repository 安装

对于 MySQL Yum Repository 支持的 Linux 发行版，请按照以下步骤安装 MySQL Shell：

  • 请执行以下任一操作：

    • MySQL Yum Repository 已经作为系统上的软件存储库，并且该存储库已配置了新的发行包 mysql80-community-release。

    • MySQL Yum Repository 已经作为系统上的软件存储库，但已使用旧的发行包 mysql-community-release 配置了该存储库，则最简单的安装方法是，使用新的 mysql-community-release 配置 MySQL Yum Repository。 为此，您需要先使用以下命令删除旧发行包：

          sudo yum remove mysql-community-release

对于启用 dnf 的系统，请执行以下操作：

            sudo dnf erase mysql-community-release

然后，按照《添加 MySQL Yum Repository》中给出的步骤安装新的发行包 mysql80-community-release。

* **MySQL Yum Repository** 没有作为系统上的软件存储库：

  • 安装 MySQL Shell 的命令：

    sudo yum install mysql-shell

对于启用 dnf 的系统，请执行以下操作：

    sudo dnf install mysql-shell

2.3 从 MySQL 开发人员专区直接下载安装 MySQL Shell

三、在 macOS 上安装

  1. 从 http://dev.mysql.com/downloads/shell/ 下载该软件包；

  2. 双击下载的 DMG 进行安装。Finder 打开；

  3. 双击 Finder 窗口中显示的 .pkg 文件；

  4. 请按照安装向导中的步骤进行操作；

  5. 安装程序完成后，弹出 DMG。（可以删除。）

四、简单使用

使用 mysqlsh 命令可直接登录到 MySQL Shell 环境，但由于未携带登录验证信息（user、host、password）等处于未连接服务状态，在内部使用除 \c \h 等简易命令外，执行其它获取服务器信息的命令，失败报错 Not Connected.

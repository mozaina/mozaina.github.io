adbshelldumpsys的用法

2020年9月2日
| 阅读

一. 打印出设备中所有service的信息

zh@zh:~$adb shell dumpsys -l (字母l)
zh@zh:~$adb shell service list

二.查询到运行的system service后，就可以在dumpsys后面加上service的名字，查看指定的service信息

zh@zh:~$adb shell dumpsys activity

a.
zh@zh:~$adb shell dumpsys activity recents | more

延伸：另一个命令：adb shell am stack list

b.

zh@zh:~$adb shell dumpsys activity activities | more

zh@zh:~$adb shell dumpsys dropbox

Usage: dumpsys dropbox [–print|–file] [YYYY-mm-dd] [HH:MM:SS] [tag]

a.

b.

zh@zh:~$adb shell dumpsys dropbox --print system_server_wtf |more

c.

zh@zh:~$adb shell dumpsys activity services | more

zh@zh:~$adb shell dumpsys battery

zh@zh:~$adb shell dumpsys display | more

延伸：

zh@zh:~$adb shell wm –help

zh@zh:~$adb shell dumpsys cpuinfo

zh@zh:~$adb shell dumpsys window -h

zh@zh:~$adb shell dumpsys window displays

延伸：

zh@zh:~$adb shell dumpsys package -h

zh@zh:~$adb shell dumpsys meminfo com.android.systemui

zh@zh:~$adb shell dumpsys usagestats

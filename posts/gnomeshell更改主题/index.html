gnomeshell更改主题

2020年9月2日
| 阅读

在开始改进这些主题时，对顶部面板右侧部分的status icon部分图标间隔缩小，应用分类里的大图标等，后来感觉这个修改应该由大家自己决定。

提示，在 Linux Deepin 安装附件里面的主题前，需进行下面的修改：修改顶部面板 Panel 高度

sudo gedit /usr/share/gnome-shell/theme/gnome-shell.css

1

|

sudo gedit / usr / share / gnome - shell / theme / gnome - shell . css

—|—

找到文件第 283 行，修改height 值，将 32px 修改为 24px。

安装主题

下载主题压缩包：点击下载

安装方法

#1,下载上面的主题包文件，解压缩；
#2,然后将文件里的全部文件复制到目录 ~/.themes 下，有的主题可能已经存在；
#3,点击“系统设置-个人-高级设置”，打开高级设置（gnome-tweak-tool）；
#4,选择“主题-Shell 主题”，选择已经安装的主题即可！

自己动手！

#1,如果觉得“应用分类”里面的图标太大——>跟我一起修改Gnome Shell 主题（2）:H 部分

#2,如果觉得顶部面板右侧图标间隔太大——>跟我一起修改Gnome Shell 主题（2）:E 部分

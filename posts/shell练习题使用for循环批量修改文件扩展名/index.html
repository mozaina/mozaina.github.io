shell练习题：使用for循环批量修改文件扩展名 **
**

**
**

说明：（1）在linux下批量修改文件名，将下图所示命令中的”_linux”去掉。 **
**

（2）使用for循环脚本。

思路：本题的基本解题思路，先进行单个文件的改名，然后再用循环实现批量改名，这是比较常规的做法，也可以用rename专业改名工具。

  1. 方法一：

说明：使用cut、sed工具

脚本如下：

#!/bin/bash

cd /test

for i in ls|grep .*.jpg

do

mv $i echo $i |cut -d '.' -f1|sed 's#_linux##'.jpg

done

说明：

脚本执行情况，如下图所示：

  1. 说明：为了进行下面的实验，我们需要对上面的文件进行数据恢复，使用cut、awk工具。

脚本如下：

#!/bin/bash

#date:2017-7-26

cd /test

for i in ls|grep .*.jpg

do

mv $i echo $i |cut -d '.' -f1|awk -F '_' '{print $1"_"$2"_"$3}'_linux.jpg

done

说明：

脚本执行情况，如下图所示：

  1. 下面我们接着使用第二种方法，批量修改文件名。

方法二：使用cut、awk工具

脚本如下：

#!/bin/bash

#date:2017-7-26

cd /test

for i in ls|grep .*.jpg

do

mv $i echo $i|cut -d '.' -f1|awk -F '_' '{print $1"_"$2"_"$3}'.jpg

done

说明：

脚本执行情况，如下图所示：

  1. 方法三：使用cut、sed工具

脚本如下：

#!/bin/bash

cd /test

for i in ls|grep .*.jpg

do

mv $i echo $i |cut -d '.' -f1|sed -r 's#(.*)_linux#\1#'.jpg

done

说明：

  1. 方法四：使用sed工具

脚本如下：

#!/bin/bash

cd /test

for i in ls|grep .*.jpg

do

mv $i echo $i|sed 's#_linux##'

done

说明：

  1. 方法五：使用rename工具

#!/bin/bash

cd /test

for i in ls|grep .*.jpg

do

echo $i|rename “_linux” “” *.jpg

done

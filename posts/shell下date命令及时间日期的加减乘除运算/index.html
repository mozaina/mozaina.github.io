shell下date命令及时间日期的加减乘除运算

2020年9月2日
| 阅读

shell下的时间戳：自1970年1月1日（00:00:00 UTC/GMT）以来的秒数。它也被称为Unix时间戳（Unix Timestam、Unix epoch、POSIX time、Unix timestamp）是从1970年1月1日（UTC/GMT的午夜）开始所经过的秒数，不考虑闰秒。

UNIX时间戳的0按照ISO 8601规范为：1970-01-01T00:00:00Z 一个小时表示为UNIX时间戳格式为：3600秒；一天表示为UNIX时间戳为86400秒，闰秒不计算。

//从2014-12-05 19:45:44到1970-1-1总共的秒数

[oracle@longleding ~]$ date -d “2014-12-05 19:45:44” +%s

1417779944

date 可以用来显示或设定系统的日期期与时间:

1.显示日期时间

%m : 月份 (01..12)
%U : 一年中的第几周 (00..53) ( 以 Sunday 为一周的第几天的情形)
%w : 一周中的第几天 (0..6)
%W :一年中的第几周 (00..53) ( 以 Monday 为几周的第几天的情形)
%x : 直接显示日期 (mm/dd/yy)
%y : 年份的最后两位数字 (00.99)
%Y : 完整年份 (0000..9999)

%X : 相当于 %H:%M:%S
%Z : 显示时区 %a : 星期几(Sun..Sat)
%A : 星期几 (Sunday..Saturday)
%b : 月份 (Jan..Dec)
%B : 月份 (January..December)
%c : 直接显示期与时间
%d : 日 (01..31)
%D : 直接显⽰⽇期 (mm/dd/yy)
%h : 同 %b
%j : 一年中的第几天 (001..366)

%H : 小时(00..23)
%I : 小时(01..12)
%k : 小时(0..23)
%l : 小时(1..12)
%M : 分钟(00..59)
%p : 显示本地 AM 或 PM
%r : 直接显示时间 (12 小时时制，格式为 hh:mm:ss [AP]M)
%s : 从 1970 年 1 月 1 日 00:00:00 UTC 到当前为为止的秒数
%S : 秒(00..61)
%T : 直接显示时间 (24 小时制）

[oracle@longleding ~]$ date +%Y-%m-%d' '%r
2019-04-03 04:16:58 PM

2.日期的加减乘除

[oracle@longleding ~]$ echo $(date --date='3 day')  //当天日期+3天
Sat Apr 6 16:05:37 CST 2019 //当天日期+3天
[oracle@longleding ~]$ echo $(date --date='3 day ago')  //当天日期-3天
Sun Mar 31 16:05:45 CST 2019 //当天日期-3天
[oracle@longleding ~]$ echo $(date --date='3 month')  //当天日期+3月
Wed Jul 3 16:05:55 CST 2019 //当天日期+3月
[oracle@longleding ~]$ echo $(date --date='3 month ago')  //当天日期-3月
Thu Jan 3 16:06:09 CST 2019 //当天日期-3月
[oracle@longleding ~]$ echo $(date --date='3 year')  //当天日期+3年
Sun Apr 3 16:06:21 CST 2022 //当天日期+3年
[oracle@longleding ~]$ echo $(date --date='3 year ago')  //当天日期-3年
Sun Apr 3 16:06:30 CST 2016 //当天日期-3年
[oracle@longleding ~]$ echo $(date --date='3 minute ')  //当天日期+3分钟
Wed Apr 3 16:09:44 CST 2019 //当天日期+3分钟
[oracle@longleding ~]$ echo $(date --date='3 minute ago')  //当天日期-3分钟
Wed Apr 3 16:04:01 CST 2019 //当天日期-3分钟
[oracle@longleding ~]$ echo $(date --date='3 hour ago')  //当天日期-3小时
Wed Apr 3 13:07:23 CST 2019 //当天日期-3小时
[oracle@longleding ~]$ echo $(date --date='3 hour')  //当天日期+3小时
Wed Apr 3 19:07:34 CST 2019 //当天日期+3小时

3.设置系统时间

date -s //设置当前时间，只有root权限才能设置，其他只能查看。

date -s 20100101 //设置成20100101，这样会把具体时间设置成空00:00:00

date -s 01:01:01 //设置具体时间，不会对日期期做更改
date -s “01:01:01 2010-01-01 ″ //这样可以设置全部时间
date -s “01:01:01 20100101″ //这样可以设置全部时间

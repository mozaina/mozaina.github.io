Linux基础常用vivimshell命令

2020年9月2日
| 阅读

一、常用shell命令

（1）ls —罗列出当前目录下的文件

    ls -a   将隐藏文件也罗列出来
    ls -l    长格式输出

（2）cd —切换目录

   cd  ~    ----   当前用户的家目录
   cd  /     -----  根目录
   cd  ..    ----   上级目录
   cd  .     ----   当前目录

gec @ ubuntu : ~ $
当前用户名 主机名 当前位置 提示符 （超级用户 root 提示符#）

  • 普通用户xxx的家目录在/home/xxx
  • 超级用户root的家目录在/root

pwd：打印当前目录

mkdir –新建目录，例：mkdir xxx

（3）其他命令

cp 例： cp ./a.c ../b.c

mv 移动文件，也可用来修改文件名。 例： mv a ~ mv 1.c 2.c

rm 例： rm b 删除文件； rm -r xxx 删除目录

二、vi/vim的基本操作

（1）命令模式

  • 输入i /a /o进入编辑模式（编辑模式按ESC退回到命令模式）
  • 输入 : 进入尾行模式

  • ZZ 保存退出

      yy —复制一行     nyy —复制n行     dd —剪切一行     ndd —剪切n行     p —粘贴     u —撤销上次操作     :5 —光标定位到第5行

（2） 尾行模式

        :w---保存
        :wq ---保存退出
        :q!-----强制退出，不保存

按ESC退回到命令模式

附：linux与win互传文件方法

第1种方法、安装vmware tools

准备一个U盘，将 VMwareTools-10.2.0-7259539.tar.gz 压缩包放入U盘，将在Ubuntu中/media/下找到该压缩包并移动到家目录下

 tar -zxvf VMwareTools-10.2.0-7259539.tar.gz

 cd vmware-tools-distrib/

 sudo ./vmware-install.pl

一路回车，直到出现 enjoy···

第2种方法、设置共享目录

虚拟机–>设置—>选项–>启用–> 添加–>选择要共享的目录

cd /mnt/hgfs/

Shell脚本的使用

2020年9月2日
| 阅读

基本介绍

用途 ：shell是一个命令行解释器，为用户提供一个向linux内核发送请求的系统及程序，用户可使用Shell执行启动、挂起和停止一些程序。

  • 脚本格式要求：
    1）脚本以#!/bin/bash开头（一般通过#进行注释）
    2）脚本需要可执行权限

    #!/bin/bash echo “hello world”

  • 常用执行方式：
    1）赋予脚本+x权限，并执行脚本
    2）直接通过sh指令执行。

基本语法

  • 变量分为系统变量和用户变量；
  • 系统变量包括：$USER, $PATH
  • 用户变量通过=定义，通过 引 用 。 A = 100 ； e c h o “ 引用。A=100 ；echo ” 引用。A=100；echo”A”
  • unset 变量名 ：删除变量

  • readonly A = 99:将A申明为一个静态变量
    定义规则 ：
    1）变量不能以数字开头
    2）等号两侧不能有空格
    3）变量名一般习惯大写
    ** 在Shell中执行命令**
    1）通过反引号执行``
    2 ）直接通过$(ls -l)执行
    设置环境变量
    1）export 变量名=变量值（将shell变量输出为环境变量）
    2）source 配置文件（让修改后的配置文件生效）
    3）echo $变量名（查询环境变量的值）例子：source /etc/profile
    注释

  • 单行注释：#内容

  • 多行注释：:< 位置参数变量

  • 常需要获取外部输入的参用，例如：MyShell .sh xx xx xx

  • 基本语法：
    1）$n: n为数字，0为本身，1~9表示第几个参数，十以上用{}，如： $(10)
    2） $*：表示将所有的参数视作一个整体
    3） $@:也表示所有参数，只是将所有参数区别开来
    4） $#:代表所有参数的个数
    预定义变量

  • shell设计者已经定义好了的变量，可以直接使用。

  • 基本语法：
    1） $$：当前进程的PID
    2） $!：后台运行的最后一个进程的PID
    3） $?：最后一次执行命令时候的返回状态。为0则正确执行，否则执行失败。
    运算符的使用

  • 基本语法：
    1）$((运算式)) 或者 $[运算式]
    2）expr m - n ：运算符之间有有空格
    3）expr \ *,/,%：乘，除，取余
    条件判断

  • 基本语法：
    [ 条件 ]：条件判断，条件两头有空格

  • 判断语句：
    1）两个整数比较
    = ：字符串比较
    -lt：小于（less than）
    -le：小于等于（less equal）
    -eq：等于
    -gt：大于（greater than）
    -ge：大于等于
    -ne：不等于
    2）文件权限比较
    -r：有读权限
    -w：有写权限
    -x：有执行权限
    3）按照文件类型判断
    -f：为常规文件（不是隐藏文件）
    -e：文件存在
    -d：为一个目录

    #!/bin/bash #判断文件是否存在 if [ -e $1 ] then echo “文件存在” fi

流程控制

  • 分支控制：

    #!/bin/bash if [ 条件判断 ] then 程序 elif [ 条件判断 ] then 程序 fi

  • case流程控制：

    #!/bin/bash case $变量名 in “值1”) 程序1 ;; “值2”) 程序2 ;; *) 其他条件 ;; esac

  • for循环

    #写法1 for 变量 in 值1 值2… do 程序 done

    #写法2 for((初始值;循环条件;变量变化)) do 程序 done

例子：

#打印命令行每个参数
for i in "$@"
do
    echo "$i"
done

#打印1~100的和
SUM=0
for((i=1;i<=100;i++))
do
    SUM=$[$SUM+$i]
done
echo "$SUM"

  • while循环

    while [ 条件判断 ] do 程序 done

  • 命令1 && 命令2：命令1成功才对执行命令2

读取控制台输入

  • read （选项） （参数）
    选项：
    -p:指定读取时的提示符
    -t：指定读取时等待时间
    参数：
    读取值的变量名

函数

  • 可以用系统函数和自定义函数。

  • 系统函数：
    1）basename [pathname] [suffix]：常用于去除完整路径中/部分，获取文件名字
    2）dirname 文件绝对路径：获取文件的路径

  • 自定义函数

    [function]funname[()] { Action; [return int;] }

    #例子 #求和 function getSum(){ SUM=$[$n1+$n2] echo “和是$SUM” } read -p “请输入第一个数字n1：” n1 read -p “请输入第二个数字n2：” n2 getSUM $n1 $n2

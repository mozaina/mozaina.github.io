Python之一怎么在Shell上运行文件

2020年9月2日
| 阅读

Python的交互命令行很方便，但是对于一个模块的执行很麻烦，因为交互执行后代码就不能用了。如果把代码放在一个文件里面，这样每次想用调用文件就可以了。
前面我不熟悉，不知道调用文件需要什么语句执行，写的时候要回头找，就直接记录下来吧。也许还有其他方法，但是先分享一下吧，大神请指点，勿喷！
首先把一段代码保存在一个文件里，例如命名为“li.py”。

# 先写一个简单的三行代码作为第一个脚本文件
print(2**100)
x='Spam'
print(x*8)

然后在交互命令行输入下面的语法就可以了。

from li import *

这是导入该模板，但好像只在第一次时执行前面的全部代码，并不是执行文件，而是导入文件。如果后面还想运行就不行了，但是可以输出个别语句和函数，比如输出“Spam”，要在下一行输入“print x”，像下面这样：

>>> from script1 import *
>>> print x
Spam

这种方法输出模板里的函数很方便，可以直接调用函数。
还有一个简单的想运行整个文件代码的方法是在编辑器里直接按F5或者点击运行，它就会运行了，运行结果是：

1267650600228229401496703205376
SpamSpamSpamSpamSpamSpamSpamSpam

这两种方法有区别，第一种是导入，第二种是直接运行。不过我现在还不很了解，这几天熟悉之后再详细地写一下。

第三部分shell脚本

2020年9月2日
| 阅读

一、第一个shell脚本

本文学习资源来自：菜鸟教程

  1. 首先创建一个test.sh文件，内容如下

    #!/bin/bash echo “hello world”

  2. 让该文件具备可执行权限

    chmod +x ./test.sh

  3. 注意，一定要写成 ./test.sh，而不是 test.sh，运行其它二进制的程序也一样，直接写 test.sh，linux 系统会去 PATH 里寻找有没有叫 test.sh 的，而只有 /bin, /sbin, /usr/bin，/usr/sbin 等在 PATH 里，你的当前目录通常不在 PATH 里，所以写成 test.sh 是会找不到命令的，要用 ./test.sh 告诉系统说，就在当前目录找。

二、变量定义及单双引号的使用

  1. 变量命名规则如下：
  • 命名只能使用英文字母，数字和下划线，首个字符不能以数字开头。
  • 中间不能有空格，可以使用下划线（_）。
  • 不能使用标点符号。
  • 不能使用bash里的关键字（可用help命令查看保留关键字）。

  1. 举例：注意只有双引号中可以应用变量，单引号是原样输出

    #!/bin/bash your_name=‘xiaoyezi’ my_name=${your_name} for file in $(ls /tmp); #遍历有命令的文件，只能用小括号()，引用自定义的变量用花括号{} do echo “tmp has ${file}” done unset my_name #单行注释：删除变量 :<<EOF 多行注释内容… 多行注释内容… 多行注释内容… EOF

三、执行脚本时，通过控制台输入传参

#!/bin/bash
echo "执行的文件名：$0"; 
#脚本内获取参数的格式为：$n,$0会指向执行的命名，所以取参从$1开始。
echo "第一个参数为：$1";
echo "第二个参数为：$2";
echo "第三个参数为：$3";

四、基本运算符

  1. 原生bash不支持简单的数学运算，但是可以通过其他命令来实现，例如 awk 和 expr，expr 最常用。
  2. 表达式和运算符之间要有空格，例如 2+2 是不对的，必须写成 2 + 2，这与我们熟悉的大多数编程语言不一样。

  3. 完整的表达式要被 `` 包含，注意这个字符不是常用的单引号，在 Esc 键下边。

    #!/bin/bash sum_num=expr 2 + 2 echo $sum_num a=10 b=20 val=expr $a + $b #乘号(*)前边必须加反斜杠()才能实现乘法运算； val1=expr $a \* $b #条件表达式 if [ $a -le $b ] then echo “a小于等于b” else echo “a大于b” fi
    #注意区分算术运算符和字符串运算符，文件运算符需要时再查询

五、定向输出信息到文件

#!/bin/bash
echo "It is a test" > myfile #会覆盖源文件
echo "It is a test" >> myfile #以追加的方式输出到文件

六、实用案例

  1. 修改文件中，以xxx开头所在行的全部内容

    #!/bin/bash #获取当前日期，格式为20200820 current_date=$(date “+%Y%m%d”) #替换a.conf文件中，以version开头的内容，注意version前有个c sed -i “/^version/cversion=${current_date}” a.conf

  2. 查找文件内容，并使用grep抓取赋值

    file_path=/tmp/a.conf node_path=server 10.11.22.33:8080 node_status=cat ${file_path} | grep "${node_path}" echo ${node_status}

  3. 进入debug调试shell脚本

    sh -x ./myshell.sh

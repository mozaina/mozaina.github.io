Windows下通过shell脚本自动截屏到脚本所在目录

2020年9月2日
| 阅读

在网上找了下自动截屏的脚本都是在Mac和linux环境下， 发现路径或者执行过程总有些问题； 现将自己写的自动截屏脚本记录在此，方便以后使用。

1.android adb 截屏命令

adb shell screencap -p ./sdcard/test.png

2.完善自动化截屏

我们通过screencap 这个命令就可以截取到Android设备当前屏幕内容, 利用这个命令我们的脚本要完善的两点就是：

  • 截屏图片以当前时间命名

  • 保存在当前目录以便查看

    #!/bin/bash

    #追踪执行日志 set -x

    FILEPATH=$(cd dirname $0; pwd)

    DATETIME=date +%Y%m%d%H%M%S

    adb root adb remount adb shell screencap -p ./sdcard/$DATETIME.png

    adb pull ./sdcard/$DATETIME.png $FILEPATH/$DATETIME.png

    exit

新建文本文件, 例如可以命名为screencap.sh, 保证以“ .sh ”为文件名后缀，并复制上面的代码到文件里保存。

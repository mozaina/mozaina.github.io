shell中的while语句与until语句的定义与示例

2020年9月2日
| 阅读

1.while语句的格式

while 条件（只有条件为真时才执行，条件为假不作任何处理）
do
done

2.while语句的示例

示例一

vim while.sh


#!/bin/bash
while ture(只有为真才执行)
do
    read -p "please input word: " WORD条件为真会一直执行，不会中断
    echo $WORD
done



sh while.sh




示例二(until语句与while语句相反)

vim until.sh


#!/bin/bash
until false(只有为假才执行)
do
    read -p "please input word: " WORD
    echo $WORD
done


sh while.sh

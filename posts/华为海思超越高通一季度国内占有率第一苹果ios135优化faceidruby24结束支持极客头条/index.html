国内要闻

阿里参股韵达2%

韵达股份发布年报信息显示，阿里巴巴已成为其股东之一，持股2%。目前，民营快递行业六大公司，顺丰、申通、圆通、中通、韵达和百世，已有五家加入“阿里系”。

中国物流学会特约研究员杨达卿表示，这是阿里看好快递行业的表现，参股韵达是多赢局面：阿里的智慧物流网络建设将进一步加速，给商家消费者提供更好更快物流服务。韵达将聚焦技术升级，加快降本提效，开拓更多新场景，搭上物流新基建快车。全社会有望迎来物流成本效率的进一步优化，对于制造业来说是利好。

董明珠回应奥克斯受罚：不合格依然举报

近期，格力举报奥克斯空调产品不合格的结果已经出来，昨日，董明珠表示，“对奥克斯的处理是必需的，这是给社会一个交代。我不仅是买了奥克斯的，我准备买各种企业的产品，拿回来进行检测，如果发现不合格，我们依然举报。我们也叫别人买我们的产品，进行检测、监督，我们要愿意去接受别人监督。这是推动社会进步，每一个人应该去做的，互相地推动行业进步。”（新京报）

华为海思麒麟芯片居一季度国内市场占有率第一，首超高通骁龙

调研机构CINNO Research的数据显示，2020年Q1，受疫情等因素的严重影响，中国智能手机出货量出现大幅下滑，由此导致手机处理器出货量同比锐减44.5%。华为海思成为中国市场唯一一家Q1出货量未出现同比下滑的品牌，市场份额为43.9%，首次超过高通，成为中国市场出货量最大的手机处理器品牌。高通、联发科、苹果分列二、三、四位。

百度云调整KPI：李彦宏要求扭亏为盈，毛利至少20%

百度云今年调整了利润指标。从知情人士获悉，百度云2020年的OKR中增添了利润指标，毛利是第一考核因素。“Robin（李彦宏）的OKR对百度云的要求就一点：扭亏为盈。”一位百度云内部人士表示，“我们收到的要求是毛利至少20%”。（ Tech星球）

国际要闻

谷歌安全团队再次公布 iOS 一系列安全漏洞

据国外媒体报道，谷歌旗下安全团队 Project Zero 日前再次为苹果公司挑错，公布了一系列可能为黑客提供入口点的基本漏洞。去年 7 月，Project Zero 团队曾公布了影响 iOS 系统的 6 个「无交互」安全漏洞。这也是谷歌 Project Zero 团队昨日发布的这份报告的主题，该报告公布了大量的新漏洞。

Lyft宣布裁员近千人：在员工总数中占17%

美国打车服务公司Lyft周三在一份监管申报文件中宣布裁员982人，还有288人将暂时休假。Lyft称，这一裁员人数在公司员工总数中所占比例为17%。另外，Lyft还已削减所谓“豁免员工”（exempt employee）的基本工资，为期12周。Lyft表示，此次降薪将从5月开始，高管领导层降薪30%，副总裁降薪20%，其他所有“豁免员工”降薪10%。（新浪科技）

苹果iOS 13.5优化Face ID：戴口罩解锁iPhone更方便

苹果开始推送iOS 13.5/iPadOS 13.5开发者预览版Beta 3更新，在iOS 13.5测试版中，苹果优化了Face ID，当iPhone检测到用户戴着口罩时，会直接弹出密码输入界面，而在这之前，带着口罩想要使用Face ID解锁比较麻烦，通常情况下，向上滑动多次才会出现密码输入界面。

这个功能目前只在苹果iOS 13.5开发者预览版Beta 3中可用，另外这个版本还带来了疫情“曝光通知”功能。（新浪数码）

谷歌Meet视频会议服务针对所有用户免费

据外媒报道，Alphabet旗下谷歌周三表示，任何用户很快都可以在Meet上主持免费视频会议，这将使这款此前仅供企业使用的工具成为Zoom和其他同类服务在冠状病毒爆发期间争夺用户的强大竞争对手。拥有1亿日活跃用户的Meet需要谷歌的商业或教育账户才能建立通话，虽然谷歌长期以来一直提供有免费版本的商务工具，包括Gmail和Google Docs，但它在三年前推出的较新服务Meet还没有类似的版本。（腾讯科技）

程序员专区

Fedora 32 发布

Fedora 发行版项目正式释出了最新的 Fedora 32，包括了工作站、服务器、CoreOS、Silverblue 和 IoT 等多个版本。Fedora 32 使用的桌面环境为 GNOME 3.36，其它重要的工具和语言更新包括 GCC 10、Ruby 2.7 和 Python 3.8。随着 Python 2 结束支持，绝大部分 Python 2 包已经从 Fedora 里移除。开发者和用户如果仍然需要可以使用 python27 包。（Solidot）

Ruby 2.4 结束支持

自去年 3 月底开始，Ruby 2.4 系列进入安全维护阶段。一年过去了，这个阶段告一段落。因此，自 2020 年 3 月 31 日起，对 Ruby 2.4 系列的支持全部结束。来自 Ruby 较新版本的缺陷和安全修正不再移植到 2.4 系列，也不再发布 2.4 的补丁版本。Ruby 开发团队于 2020 年 3 月 31 日发布了 Ruby 2.4.10，但这是出于给用户一点时间来迁移到更新的版本。强烈建议尽快升级到更新的版本。（开源中国）

androidshell脚本基础2

2020年9月2日
| 阅读

shell脚本基础1中我们只是一些菜鸟教程里的一些基础教程

在这里我们要利用shell脚本操作我们的服务器

同样我们操作的文件是.sh结尾的

比如我们想要将我们的服务器jdk进行配置，我们知道首先上传.tar的压缩包到我们的服务器上，然后我们在该目录下进行解压，然后在、etc/profile/里面进行添加环境的配置，然后我们输入Java -version进行验证，我们用shell脚本怎么写呢，附上代码

#!/usr/bin/expect
#登录ssh
#通过ssh 服务器 spawn与expect 配合使用以及send配合使用
spawn ssh root@ip
#期望值
expect “password:”
#输入密码 点击回车\n
send “xxxx\n”
#期望
expect “iZ2ze7npzhr9nqpzetj4kwZ:”
#输入ls /
send “ls /\n”
#期望
expect “iZ2ze7npzhr9nqpzetj4kwZ:”
#输入
send “cd /usr/local\n”
#期望
expect “/usr/local”
#输入 ls
send “tar -xvf jdk1.8.0_151.tar\n”
#期望
expect “/usr/local”
#向文件里输入内容
send “cd /etc\n”
#期望etc
expect “/etc”

#向profile里面输入配置环境
send “ echo export JAVA_HOME=/usr/local/jdk1.8.0_151 >> profile\n”
#期望etc
expect “/etc”
#向profile里面输入配置环境
send “ echo export CLASSPATH=.:/usr/local/jdk1.8.0_151/lib/dt.jar:/usr/local/jdk1.8.0_151/lib/tools.jar >> profile\n”
#期望etc
expect “/etc”
#向profile里面输入配置环境
send “ echo export PATH=/usr/local/jdk1.8.0_151/bin:\$PATH >> profile\n”
#输入系统重新加载配置文件
#期望etc
expect “/etc”
#系统重新加载配置文件
send “source profile\n”
#输入Java -version
send “java -version\n”
#留在服务器
interact

注意：我们在终端找到这个.sh文件后输入expect.sh进行操作,上面的ip和密码自己写

下面我们说一下图片上传的shell脚本的编写：

#!/usr/bin/expect
#将文件上传上去，同时登陆服务器
#开启新进程命令 spawn
spawn scp /Users/yuliguo/Desktop/a.jpg root@ip:/home/
#期望值
expect “password:”
#输入密码
send “xxxxx\n”
#期望值
expect “yuliguo”
spawn ssh root@ip
#期望值
expect “password:”
#输入密码
send “xxxxx\n”
#留在服务器里面

interact

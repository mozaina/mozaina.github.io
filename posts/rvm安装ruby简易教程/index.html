rvm安装ruby简易教程

2020年9月2日
| 阅读

安装前的准备

下载工具

sudo apt-get install curl git

下载

curl -L get.rvm.io | bash -s stable

加载rvm

source ~/.rvm/scripts/rvm

安装ruby

查看可以使用 的ruby

rvm list known

安装

rvm install 2.2.1
—此处选择一个版本号就行了需要等待一些时间—

错误

很多人写过错误，但我还是没找到合适的，于是就一个一个试试，最终解决了”自己的问题“
注明：本平台是ubuntu14.04
此时rvm和ruby还不能用，比如rvm use 2.2.1 –default会提醒你rvm不是个函数，再比如ruby -v没有版本信息
于是http://stackoverflow.com/questions/22773693/rvm-zsh-rvm-is-not-a-function- selecting-rubies-with-rvm-use-will-not-w
这个网页的第二条会告诉你source HOME/.rvm/scripts/rvm，但是系统重启后还是不行，你还得每次重新source HOME/.rvm/scripts/rvm
再rvm use 2.2.1 –default
然后，有人告诉你，在~/.bash_profile后面加上

[[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm" # Load RVM into a shell session*as a function*

试了试，依然没用，先说下该文件的作用
~/.bash_profile是用户进入的时候调用，同时还会调用~/.bashrc和~/profile,但是上述代码在~/.bash_profile中已经有了，并且我们的问题是重启后仍然需要重新设置，于是就把

source $HOME/.rvm/scripts/rvm
rvm use 2.2.1 --default

写到了/etc/profile里面，好处是开机就加载。当然需要root权限，
当然也许也能写在~/.bashrc或其他用户文件里面，只是写在~/.bash_profile里面竟然没用于是就不试了。

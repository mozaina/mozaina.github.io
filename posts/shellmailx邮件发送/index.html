shellmailx邮件发送

2020年9月2日
| 阅读

查看是否能够使用mailx命令

[root@sms wqbin]# mailx -V
12.4 7/29/08

如果没有命令，先下载按照mailx

wget http://ftp.debian.org/debian/pool/main/h/heirloom-mailx/heirloom-mailx_12.5.orig.tar.gz
tar zxvf heirloom-mailx_12.5.orig.tar.gz    //解压
cd heirloom-mailx-12.5/
make && make install UCBINSTALL=/usr/bin/install     //安装

配置mailx文件mail.rc

[root@sms wqbin]# vim /etc/mail.rc   (在配置文件最后添加)
set from=sendmail@mail.com                      #收件人显示的发件人名称，可填写你的名字等
set smtp=smtp.mail.com                               #你所使用的外部邮箱的smtp服务器地址
set smtp-auth-user=sendmail@mail.com     #所使用的外部邮箱的用户名
set smtp-auth-password=123456                 #你所使用的外部邮箱密码
set smtp-auth=login                                      #不需要修改

命令格式

Usage: mailx -eiIUdEFntBDNHRV~ -T FILE -u USER -h hops -r address -s SUBJECT -a FILE -q FILE -f FILE -A ACCOUNT -b USERS -c USERS -S OPTION users


 mailx [-BDdEFintv~] [-s subject] [-a attachment ] [-c cc-addr] [-b bcc-addr] [-r from-addr] [-h hops] [-A account] [-S variable[=value]] to-addr . . .
       mailx [-BDdeEHiInNRv~] [-T name] [-A account] [-S variable[=value]] -f [name]
       mailx [-BDdeEinNRv~] [-A account] [-S variable[=value]] [-u user]

命令示例

shell 中示例

echo “邮件内容” | mailx -v -s “邮件标题” 收件人1@mail.com,收件人2@mail.com

或者

直接将文件中的内容作为邮件内容发送

mailx -v -s “邮件标题” 收件人1@mail.com,收件人2@mail.com < test.txt

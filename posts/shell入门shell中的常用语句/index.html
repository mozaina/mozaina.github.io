expect

expect是自动应答命令用于交互式命令的自动执行
spawn是expect中的监控程序，其运行后会监控命令提出的交互问题
send 发送问题答案给交互命令
“\r” 表示回车
exp_continue 标示当问题不存在时继续回答下面的问题
expect eof 标示问题回答完毕退出expect环境
interact 标示问题回答完毕留在交互界面
set NAME [ lindex $argv n ] 定义变量，从0开始计数

首先需要下载 expect软件

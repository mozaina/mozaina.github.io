珍藏多年的shell脚本

2020年9月2日
| 阅读

ps -l
查看进程

ps -ef | grep java
jps -lv
查看jvm参数

ssh从服务器上下载文件
scp username@servername:/path/filename /var/www/local_dir（本地目录）

ssh上传本地文件到服务器
scp /path/filename username@servername:/path

nmon -s 2 -c 150 -F test.nmon
nmon5分钟监控，生成文件

nicstat -i [网卡名称] [间隔时间]
网卡饱和度

df -h
查看挂载使用情况

find -type f|xrags grep ‘字符串’
查找包含‘字符串’的文件的文件名

grep ‘字符串’ -r
查找包含‘字符串’的文件的文件名

arch
查看处理器架构

grep 字符串 文件名|wc -l
统计字符串在文件中出现的次数，一行只统计一次

cat /dev/null > /etc/test.txt
清空文件

find /home/ate/ -type f | xargs rm -rf
删除目录下所有文件，而不删除目录和子目录

find /home/app -ctime +7 -name “*.tar” -print | xargs rm -f
找到7天之前/home/app下生成或修改的.tar文件并删除它

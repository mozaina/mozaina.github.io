脚本创建用户1

这个脚本对于文件中存在的字符进行查找 所以创建有些不存在的用户但有相同字符串存在文本中的时候 比如 user 会出现错误

这个之可以创建如123424513 jaklfj等等的用户

#!/bin/bash
cat /etc/passwd|grep $1 >/dev/null && {
echo “user exit”
exit 1
}
useradd $1

echo $2|passwd $1 –stdin

使用如下脚本可以完整的创建用户

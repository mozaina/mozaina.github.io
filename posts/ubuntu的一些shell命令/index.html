Ubuntu的一些shell命令

2020年9月2日
| 阅读

压缩与解压

.tar
解包：tar xvf FileName.tar
打包：tar cvf FileName.tar DirName
（注：tar是打包，不是压缩！）
---------------------------------------------
.gz
解压1：gunzip FileName.gz
解压2：gzip -d FileName.gz
压缩：gzip FileName
.tar.gz 和 .tgz
解压：tar zxvf FileName.tar.gz
压缩：tar zcvf FileName.tar.gz DirName
---------------------------------------------
.bz2
解压1：bzip2 -d FileName.bz2
解压2：bunzip2 FileName.bz2
压缩： bzip2 -z FileName
.tar.bz2
解压：tar jxvf FileName.tar.bz2
压缩：tar jcvf FileName.tar.bz2 DirName
---------------------------------------------
.bz
解压1：bzip2 -d FileName.bz
解压2：bunzip2 FileName.bz
压缩：未知
.tar.bz
解压：tar jxvf FileName.tar.bz
压缩：未知
---------------------------------------------
.zip
解压：unzip FileName.zip
压缩：zip FileName.zip DirName
---------------------------------------------
.rar
解压：rar x FileName.rar
压缩：rar a FileName.rar DirName

磁盘

在终端输入df -h显示磁盘使用情况

使用 du查看某个文件或目录占用磁盘空间的大小
du -ah –max-depth=1 这个是我想要的结果 a表示显示目录下所有的文件和文件夹（不含子目录），h表示以人类能看懂的方式，max- depth表示目录的深度。

du -sh : 查看当前目录总共占的容量。而不单独列出各子项占用的容量

查看安卓服务列表

service list

查看电脑配置

在窗口中输入【free -m】按enter键执行
然后会在下面第一行的【mem:】后面的【total】下面显示你的内存总大小。

需要先在终端输入【df】，显示的sd1就是我当前的磁盘。后面的第一个数据就是磁盘大小。

在终端输入【cat /proc/cpuinfo】按enter执行后则显示出CPU的型号之类的信息.

命令是【lspci |grep VGA】按enter执行,然后在下面的一行中，最后面会显示你的显卡型号.

# uname -a               # 查看内核/操作系统/CPU信息
# env                    # 查看环境变量
# df -h                     # 查看各分区使用情况
# uptime                 # 查看系统运行时间、用户数、负载 
# cat /proc/loadavg         # 查看系统负载
# ps -ef                    # 查看所有进程 
# top                    # 实时显示进程状态
# chkconfig --list          # 列出所有系统服务 
# chkconfig --list | grep on   # 列出所有启动的系统服务

防火墙关闭

Ufw disable
/etc/init.d/ufw stop/restat/start

查看服务运行状态

Netstat -a | grep servicename 输出空说明未开启

安装软件

Apt-get install softname

删除软件

Apt-get remove softname

修改ip等

Vim /etc/network/interfaces

Ubuntu重启

Reboot

查找特定的句柄在文件中

Find filename -name *.c | xargs grep “temp”

查看安卓版本信息

Cat /android/build/core/version_default.mk中PLATFORM_SDK_VERSION=x.x.x

查看Ubuntu版本信息

Cat /etc/issue uname -a

使用ctags实现跳转

Ctags -R
Vim /etc/vim/vimrc
Set tags=/folder/tags

查看所有用户

Cat /etc/shadow

Mm与mmm命令

Source /android/build/envsetup.sh
在有Android.mk文件的目录下执行这些命令来实现编译

安卓与Ubuntu的文件传输

Adb +push +Ubuntu文件 +android目录

使用samba与windows共享文件

修改/etc/smaba/smb.conf配置文件，添加一个共享目录

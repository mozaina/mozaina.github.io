今天学习了js shell的相关知识

一、怎么开始？

有两种办法可以开始我们js shell的配置：

1.在所给链接 http://ftp.mozilla.org/pub/firefox/nightly/latest- oak/下载需要的安装包并解压，直接打开js命令文件即可在交互模式下使用shell执行js代码（出现js提示符就可以进行编写js代码）；

2.用 ‘brew install spidermonkey’ download spidermonkey，就可以直接用‘js + 文件名’ 的方式用shell执行。

注意：此时不需要再用‘js’进入到js环境，再执行命令了，直接在目录下执行‘js + 文件名’即可。

二、讲讲为什么要用 Javascript Shell

**
**

js在浏览器就可以直接运行，但是随着js的发展，它不仅可以作为桌面执行程序，还可以作为服务器（nodejs），这种情况下Mozilla提供了js shell这样的编程环境。但事实上，我们想要通过JavaScript学习基本数据结构和算法的话，随便打开任意console（控制台）就可以达到我们学习的目的，本人是个遇到问题从不逃避的人，所以问同学Google查资料等等想尽一切办法弄清楚为啥我会犯错，并不再犯同样的毛病。第一，提高学习能力；第二，技多不压身，了解新知识本身就很吸引人。

Shell笔记流程控制ifcaseforwhilebreakcontinue

2020年9月2日
| 阅读

if语句

语法：
if - else

if [ 条件判断式 ]
then
    command1
    command2
    ...
else
    command3
    ...
fi

if - elif - else

# if - elif - else
if [ 条件判断式1 ]
then
    commandA
    ...
elif [ 条件判断式2 ]
then
    commandB
    ...
else
    commandC
    ...
fi

说明：if为流程控制的开始，fi为结束。
注意：
1.空格，if后面和中括号的条件表达式中均有空格。
2.shell的流程控制过程中，满足if的条件时，command语句必须存在，不能为空；满足else的条件时，command语句也不能为空，如为空则删除else部分。

如：

a=1
b=2
if [ $a == $b ]
then
    echo "a=b"
elif [ $a -le $b ]
then
    echo "a **=b"
fi**

这里遇到个问题：第 6 行: [: 缺少 `]’，原因是汉字的空格。

case语句

说明：case语句为选择语句。case语句中一个值匹配一种流程。
语法：

case 值 in
模式1)
    commandA
    ...
    ;;      表示命令的结束，相当于break
模式2）
    commandB
    ...
    ;;
模式3）
    commandC
    ...
    ;;
模式4）
    commandD
    ...
    ;;
esac

如：

read number
echo "输入值为: $number"
case number in
1)
    echo "1"
    ;;
2）
    echo "2"
    ;;
3)
    echo "3"
    ;;
4)
    echo "4"
    ;;
esac

for循环

语法：

for (( 初始值;循环条件;变便变化 ))
do 
    command
done

如：

#打印1-10的数字。
s=0
for (( x=0;x<=10;x++))
do
    echo $x
done

for无限循环

for(( ; ; ))

while循环

语法：

 while [ 条件判断式 ]
 do
    command
 done

如：

#打印1-10的数
x=0
i=1
while [ $x -le 10 ]
do
    echo $x
    x=$[ $x+1 ]
done

while无限循环

while true
do
    command
done

跳出循环

break
说明：break用于中断当前控制流程下的所有循环。

x=0
i=1
while [ $x -le 100 ]
do
    echo $x
    x=$[ $x+1 ]
    let y=x\*x
    if [ $y -ge 99  ]
    then
        echo "跳出循环"
        break
    fi
done

contiune

说明：跳过本次循环，执行下一次循环。
如：

x=0
i=1
while [ $x -le 100 ]
do
    echo $x
    x=$[ $x+1 ]
    let y=x\*x
    if [ $y -ge 99  ]
    then
        echo "大于99"
        contiune
    fi
done

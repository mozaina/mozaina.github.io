win10下rubyonrails环境搭建

2020年9月2日
| 阅读

下载VMware12并安装 网上有Key

下载Ubuntu16.04.iso

打开VMware 新建一个空白虚拟机

右键新建的虚拟机-CD/DVD-使用ISO映像-Ubuntu16.04.ISO

开机-选择简体中文-安装跟着向导走

其中硬盘方案

20G下

VMware: ALT+左键移动窗口

类型交换空间(swap) 4096M

/boot 512M

/ 10240M

/home 剩余

设置ROOT: sudo passwd root

获取ROOT: su

打开终端: 桌面空白处 CTRL+ALT+T

vmtool 安装:

右键虚拟机-设置-CD/DVD-连接-使用物理驱动器

开机

把DVD中文件COPY到桌面

在文件目录右键终端 tar -xzvf VMwareTools-9.6.1-1378637.tar.gz

进入 vmware-tools-distrib 文件夹

右键终端 sudo ./vmware-install.pl

之后一路enter直到结束

右上角设置-系统设置-亮度和锁屏-从不-关闭-取消唤醒密码

设置共享目录

先关机-虚拟机-设置-选项-共享文件夹-总是启用-添加-选择我在windows新建的VMshell文件夹 -确定

共享文件夹在Ubuntu的文件管理器-计算机-mnt-hgfs中

桌面右键终端 ln -s /mnt/hgfs/VMshell /home/登录用户名/桌面

http://blog.csdn.net/kurosakimaon/article/details/53575301

http://blog.csdn.net/theusprome/article/details/52927870

输入法-浏览器搜索搜狗下载搜狗for linux

登陆用户密码框并不强制英文输入,有可能还是系统的中文输入法

进入文件所在文件夹-更改文件名为sogou.deb-右键终端

sudo apt install libopencc1 fcitx-libs fcitx-libs-qt

sudo dpkg -i sogou.deb

注销重新进入

右上角键盘-搜狗拼音-左键点一下桌面空白

右上角键盘-设置-高级-最底下Fcitx设置

搜狗上移到顶-删除2个不要的-下移符号右边符号-确认-关闭对话框-应用-关闭对话框

环境搭建

桌面右键终端

http://happypeter.github.io/rails10/02_getrails.html

全部复制粘贴

环境搭建完毕!!!

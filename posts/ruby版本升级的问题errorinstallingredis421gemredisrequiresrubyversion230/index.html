图解

安装redis-4.2.1.gem出现Ruby版本问题

[root@emooco redis-cluster]# gem install redis-4.2.1.gem
ERROR:  Error installing redis-4.2.1.gem:
    redis requires Ruby version >= 2.3.0.

从错误提示可以看的出来这个版本太low，因此，我们要升级版本。

查看当前ruby的版本命令

命令

[root@emooco redis-cluster]# ruby -v
ruby 2.0.0p648 (2015-12-16) [x86_64-linux]

发现是2.0.0版本的，远远不能满足呀。

安装RVM的命令

命令

[root@emooco ~]# gpg --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 94558F59
[root@emooco ~]# curl -L get.rvm.io | bash -s stable

查询RVM的安装并打印

命令

[root@emooco ~]# find / -name rvm -print

[root@emooco ~]# source /usr/local/rvm/scripts/rvm

删除当前ruby的版本

命令

[root@emooco ~]# rvm remove 2.0.0

从以上的额版本中选择一个安装【我选最新的2.7.0】

命令

[root@emooco ~]# rvm install 2.7.0

一次安装不成功多安装两次。

选择安装的ruby版本并设为默认

命令

[root@emooco ~]# rvm use 2.7.0 --default

再次查看ruby的版本【我们想要的最新版本2.7.0】

命令

[root@emooco ~]# ruby -v
ruby 2.7.0p0 (2019-12-25 revision 647ee6f091) [x86_64-linux]

安装redis-ruby

命令

[root@emooco ~]# gem install redis-4.2.1.gem

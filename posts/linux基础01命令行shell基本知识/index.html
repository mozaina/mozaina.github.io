什么是Shell

操作系统底层有个内核(Kernel)，内核用来实现计算机操作系统底层基本功能，比如：磁盘读写、网络的连接、键盘鼠标的驱动、图形界面等等。但是用户无法直接操控内核(Kernel)。

这时候就需要一个“Shell”程序，Shell将用户的指令翻译成内核可以理解的信息。

Shell的分类

  • CLI: Command Line Interface —— 命令行界面
  • GUI: Graphical User Interface —— 图形界面

Linux的Shell

  • CLI: BASH
  • GUI: GNOME

提示符

  • #: 代表以根用户(root)登录
  • $: 代表以普通用户登录

命令一般由三个部分组成

  • 命令
  • 选项
  • 参数

&符号

  • 任何命令后加上 &，代表着将这条命令放在后台运行
  • 使用jobs命令可以查看后台运行的作业

BASH——历史记录

  • 使用上下键查看之前的命令
  • 使用 history命令 查看全部历史记录

切换用户

  • su -: 切换到root用户并使用一个新的运行环境

  • sudo: 使用管理员用户身份运行命令

显示当前用户信息: id
修改当前用户密码: passwd

uname

uname
只返回Linux

uname -r
返回内核版本号

uname -a
uname --all
返回所有系统信息

cat

cat指令用于查看文件的内容

windows10系统下安装配置ruby开发环境

2020年9月2日
| 阅读

1. 前言

由于ruby在windows下不友好

因此考虑使用wsl来搭建环境，wsl是“Windows Subsystem for Linux”的缩写，顾名思义，wsl就是Windows系统的Linux子系统

使用wsl搭建环境后，使用vscode作为开发环境

以下是搭建过程

2. 配置wsl

可以参考下面这篇文章

WSL 使用指南——02 安装配置

3. 配置vscode

打开vscode，一般vscode检测电脑存在有wsl环境后，会自动让你装一个插件Remote - WSL，安装这个插件即可

安装完之后

点击右下角的按钮，在弹出来的框中，选择new Window，然后就会出来一个ubuntu的vscode开发环境

4. 安装ruby

直接参考官网ubuntu的安装流程

如果是其他linux发行部，可以去看官网上的安装方式

# 更新apt-get包
sudo apt-get update
# 安装ruby
sudo apt-get install ruby-full

然后根据提示一步步操作就可以啦

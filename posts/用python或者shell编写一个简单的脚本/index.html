用python或者shell编写一个简单的脚本

2020年9月2日
| 阅读

今天去面试被一个笔试题难住了，题目是：
用过python或者perl或shell写过程序或脚本吗？请用自己熟知的程序，写1-3个自己熟知的脚本。
这个题我没写，因为不会就空着了。回来之后百度了一下，现在记下来当笔记。
1、使用python语言：(方法很多，下面是在linux系统下编写)

  1. vim helloworld.py （新建并编辑一个文件）

  2. #!/usr/bin/env python
    print(‘helloworld’)

  3. chmod a+x helloworld.py（给文件赋权限，把普通文件变成可执行文件）

  4. ./helloworld.py （启动脚本）

    也可在windows环境下直接新建一个.py文件，输入print(‘helloworld’) ，保存，双击运行即可
    2、使用perl语言(听都没听过，在这里就不写了)
    3、使用shell语言：

  5. vim hello.sh （新建并编辑一个文件）

    2)#！/bin/bash
    clear;
    echo ‘helloworld’;

  6. chmod 777 hello.sh （给文件赋权限，把普通文件变成可执行文件）

  7. ./hello.sh （启动脚本）

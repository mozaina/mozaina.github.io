代码块 #!/bash/bin #!filename=inbash.sh #查看文件是否含有#！ count=`grep "^#!" $1 | wc -l` if [ ${count} == 0 ]; then #定界符 EOF后的文件前面不能有空格 cat >> $1 << EOF #!/bin/bash #author: #date: `date +"%Y-%m-%d"` #description: EOF fi 调试脚本：sh -x inbash.sh out.txt 脚本讲解：先查看文件是否以#!开头，如果不是的话，就在文件中把EOF中的文本插入到文件中。 相关内容： 1.反引号“=$(): 如 count=`grep "^#!" $1 | wc -l)`和 count= $(grep "^#!" $1 | wc -l)是一样的效果 2.定界符EOF后的文本不能出现空格 3.vi删除所有的内容可使用dG 4.$1代表输入的一个参数 5.脚本不加可执行权限，可以直接用sh inbash.sh执行即可……

问：提示用户输入一个小于100的整数，求出1到该数的奇数和与偶数和 方法一 奇偶各自循环 #!/bin/bash read -p "请输入一个小于100的整数：" num //read -p 提示交互信息，变量值由用户键盘输入 sum1=0 //初始值定为0 sum2=0 if [[ $num -ge 1 && $num -lt 100 ]];then //当输入数大于等于1小于100时 for ((i=0; i<=$num; i+=2)) //初始值为0，每循环一次，i加2，为偶数 do let sum1=$sum1+$i //每循环一次,总和加i，总和为偶数和 done echo "1到$num的偶数和为$sum1" //输出偶数和 for ((n=1; n<=$num; n+=2)) //初始值为1，每循环一次，n加2,为奇数 do let sum2=$sum2+$n //每循环一次，总和加n，总和为偶数和 done echo "1到$num的奇数和为$sum2" //输出奇数和 else echo "输入错误" //输入别的不符合要求的内容，提示错误 $0 //程序重新开始 fi 运行结果 [root@host opt]# ./num3.sh 请输入一个小于100的整数：6 1到6的偶数和为12 1到6的奇数和为9 [root@host opt]# ./num3.sh 请输入一个小于100的整数：999 输入错误 请输入一个小于100的整数：ii 输入错误 请输入一个小于100的整数：77 1到77的偶数和为1482 1到77的奇数和为1521 小技能：……

shell小技巧一修改一批文件后缀

2020年9月2日
当前目录下后缀为sh的文件，改为后缀shell。 这里列出两种方法，先看第一种。 方法1： #!/bin/bash str=find ./ -name \\*.sh for i in $str do mv $i ${i%sh}shell done 思路很简单，用find找到符合的文件，存如数组。然后使用mv修改文件名。 其中${i%sh}shell是修改后缀后的文件名。%是将字符串变量i中sh以及sh右侧的字符全部删除。 也可以这样写，${i%.*}.shell，删除最后一个“.”以及右侧的所有字符。 还有一种直接替换,${i/.sh/.shell/}，不过如果文件名中如果还含有“.sh”会出问题，因为他只替换第一个找到的字串，所以不精确。 因为此题后缀替换前后有特殊性，还可以这样写：${i}ell。 方法2： #!/bin/bash str=find ./ -name "f*.sh" for i in $str do mv $i $( echo $i | sed ’s#.sh$#.shell#g’ ) done 此方法主要是用了sed命令，原理跟${i/sh/shell/}类似，只不过它指明了替换结尾的字串，能精确匹配要求。……

解决XShell6强制更新的问题

2020年9月2日
最近打开X-shell 6竟然要求强制更新，否则无法打开程序， 程序提示：”要继续使用此程序，您必须应用最新的更新或使用新版本” 但是更新又无法连接服务器……总之，程序是用不了了！！ 没办法了，网上问下度娘吧。。。。。 此处省略半小时搜索与尝试……… 最终，解决办法如下，本人亲测成功！（但是如果有条件还是希望大家支持正版，毕竟人家做个产品也不易） 1. 找到程序安装的目录：我的是C:\Program Files (x86)\NetSarang\Xshell 6，安装目录可能不同找你自己的安装目录即可 2.找到安装目录下的 nslicense.dll 文件 3.需要下载一个软件c32asm（注意：这个软件是绿色免安装的，但不要放到中文目录下，不然不能用）， 可以到我的下载里找，也可以去下面的网盘下。 4.用c32asm以16进制形式打开nslicense.dll 文件 5.搜索：7F0C81F98033E1010F8681，把86改成83，保存即可 6.将保存后的nslicense.dll 文件替换到安装目录即可，打开看看吧，可以用了 以上方法来源于网络，感谢各位网友的分享。 链接：https://pan.baidu.com/s/1AQyb0brMxfhQaZCr0LGKzQ 提取码：yecp 简码笔记，让你的代码更加简约精炼。……

解决在centos7中执行shell脚本找不到命令或文件的问题

2020年9月2日

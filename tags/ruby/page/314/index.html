2020年9月2日
一、执行步骤 1、wget http://www.ruby-lang.org/en/downloads/ 下载最新版本源码，tar -zxvf 解压。 2、cd ruby-2.5.1目录，执行 ./configure 、make 、makeinstall等命令。 3、ruby -v可查看到版本信息。 4、执行gem install –no-rdoc –no-ri tprov，报错如下： ERROR: Loading command: install (LoadError) cannot load such file – zlib ERROR: While executing gem … (NoMethodError) undefined method `invoke_with_build_args’ for nil:NilClass 二、找解决方法 1、cd ruby-2.5.1/ext/zlib目录，执行 ruby ./extconf.rb、make 、make install命令。 2、make时报错（chmod +x Makefile）:make: *** No rule to make target /include/ruby.h', needed byzlib.o’. Stop. 3、更改Makefile文件， zlib.o: $(top_srcdir)/include/ruby.h 改成 zlib.o: ../../include/ruby.h；到这里就可以make成功了。 4、进入/ruby-2.5.1/ext/openssl目录执行 ruby ./extconf.rb、make 、make install命令。……

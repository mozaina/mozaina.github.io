Linux下快速安装Ruby

2020年9月2日
这里使用 RVM (命令行工具，可以提供一个便捷的多版本 Ruby 环境的管理和切换)实现在Linux系统上快速安装Ruby。 0.安装RVM： $ gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 $ \curl -sSL https://get.rvm.io | bash -s stable $ source ~/.bashrc $ source ~/.bash_profile 1.修改RVM的Ruby安装源提高安装速度： $ echo "ruby_url=https://cache.ruby-china.org/pub/ruby" > ~/.rvm/user/db 2.安装Ruby： 列出可供安装的Ruby版本： rvm list known 选择一个版本进行安装： rvm install X.X.X --disable-binary 确定Ruby是否安装成功(查询已安装的版本）： rvm list 完成……
